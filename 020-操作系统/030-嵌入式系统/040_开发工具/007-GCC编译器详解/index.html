    GCC编译器详解¶

    概述¶

    GCC（GNU Compiler Collection）是最流行的编译器套件，支持C、C++、Fortran等多种语言。

    GCC编译流程¶

    Text Only
    1
    源代码(.c) -> 预处理(.i) -> 编译(.s) -> 汇编(.o) -> 链接(可执行文件)

    基本编译¶

    单文件编译¶

    # 编译并运行
gcc hello.c -o hello
./hello

# 分步编译
gcc -E hello.c -o hello.i    # 预处理
gcc -S hello.i -o hello.s    # 编译为汇编
gcc -c hello.s -o hello.o    # 汇编为目标文件
gcc hello.o -o hello         # 链接

# C++编译
g++ hello.cpp -o hello

    Makefile集成¶

    Makefile
     1
 2
 3

 9

    CC = gcc
CFLAGS = -Wall -O2
LDFLAGS = -lm

SRCS = $(wildcard *.c)
OBJS = $(SRCS:.c=.o)

program: $(OBJS)
    $(CC) $(OBJS) $(LDFLAGS) -o $@

%.o: %.c
    $(CC) $(CFLAGS) -c $< -o $@

clean:
    rm -f $(OBJS) program
