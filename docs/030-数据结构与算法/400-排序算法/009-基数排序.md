# 基数排序

## 概述

基数排序通过按位排序实现整数排序，从最低位到最高位依次排序，使用稳定排序（计数排序）作为子过程。

## 算法思想

1. 找到最大数的位数
2. 从最低位开始，按当前位进行计数排序
3. 依次处理更高位，直到最高位

## 算法演示

```
初始: 170  45  75  90  802  24  2  66

按个位排序:
170  90  802  2  24  45  75  66

按十位排序:
802  2  24  45  66  170  75  90

按百位排序:
2  24  45  66  75  90  170  802
```

## 基本实现（LSD）

```c
int getMax(int arr[], int n) {
    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
    }
    return max;
}

void countSort(int arr[], int n, int exp) {
    int *output = (int*)malloc(n * sizeof(int));
    int count[10] = {0};
    
    for (int i = 0; i < n; i++) {
        count[(arr[i] / exp) % 10]++;
    }
    
    for (int i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }
    
    for (int i = n - 1; i >= 0; i--) {
        int digit = (arr[i] / exp) % 10;
        output[count[digit] - 1] = arr[i];
        count[digit]--;
    }
    
    for (int i = 0; i < n; i++) {
        arr[i] = output[i];
    }
    
    free(output);
}

void radixSort(int arr[], int n) {
    int max = getMax(arr, n);
    
    for (int exp = 1; max / exp > 0; exp *= 10) {
        countSort(arr, n, exp);
    }
}
```

## MSD实现

```c
void radixSortMSD(int arr[], int n, int exp) {
    if (exp == 0 || n <= 1) return;
    
    int *buckets[10];
    int count[10] = {0};
    
    for (int i = 0; i < 10; i++) {
        buckets[i] = (int*)malloc(n * sizeof(int));
    }
    
    for (int i = 0; i < n; i++) {
        int digit = (arr[i] / exp) % 10;
        buckets[digit][count[digit]++] = arr[i];
    }
    
    int index = 0;
    for (int i = 0; i < 10; i++) {
        if (count[i] > 0) {
            radixSortMSD(buckets[i], count[i], exp / 10);
            for (int j = 0; j < count[i]; j++) {
                arr[index++] = buckets[i][j];
            }
        }
    }
    
    for (int i = 0; i < 10; i++) {
        free(buckets[i]);
    }
}
```

## 处理负数

```c
void radixSortWithNegative(int arr[], int n) {
    int *positive = (int*)malloc(n * sizeof(int));
    int *negative = (int*)malloc(n * sizeof(int));
    int posCount = 0, negCount = 0;
    
    for (int i = 0; i < n; i++) {
        if (arr[i] >= 0) {
            positive[posCount++] = arr[i];
        } else {
            negative[negCount++] = -arr[i];
        }
    }
    
    radixSort(positive, posCount);
    radixSort(negative, negCount);
    
    int index = 0;
    for (int i = negCount - 1; i >= 0; i--) {
        arr[index++] = -negative[i];
    }
    for (int i = 0; i < posCount; i++) {
        arr[index++] = positive[i];
    }
    
    free(positive);
    free(negative);
}
```

## C++ 实现

```cpp
void radixSort(std::vector<int>& arr) {
    if (arr.empty()) return;
    
    int maxVal = *std::max_element(arr.begin(), arr.end());
    
    for (int exp = 1; maxVal / exp > 0; exp *= 10) {
        std::vector<int> output(arr.size());
        int count[10] = {0};
        
        for (int num : arr) {
            count[(num / exp) % 10]++;
        }
        
        for (int i = 1; i < 10; i++) {
            count[i] += count[i - 1];
        }
        
        for (int i = arr.size() - 1; i >= 0; i--) {
            int digit = (arr[i] / exp) % 10;
            output[--count[digit]] = arr[i];
        }
        
        arr = output;
    }
}
```

## 复杂度分析

| 情况 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 所有 | O(d(n + k)) | O(n + k) |

d = 最大位数，k = 基数（通常10）

## 稳定性

基数排序是**稳定排序**。

## LSD vs MSD

| 特性 | LSD | MSD |
|------|-----|-----|
| 排序方向 | 低位到高位 | 高位到低位 |
| 实现复杂度 | 简单 | 复杂 |
| 空间效率 | 较低 | 较高 |
| 递归 | 不需要 | 需要 |

## 应用场景

1. **整数排序**：大范围整数
2. **字符串排序**：字典序排序
3. **日期排序**：年月日分别排序
4. **IP地址排序**：分段排序

## 字符串基数排序

```c
void radixSortString(char *strs[], int n, int maxLen) {
    for (int pos = maxLen - 1; pos >= 0; pos--) {
        char **output = (char**)malloc(n * sizeof(char*));
        int count[256] = {0};
        
        for (int i = 0; i < n; i++) {
            char ch = pos < strlen(strs[i]) ? strs[i][pos] : 0;
            count[(unsigned char)ch]++;
        }
        
        for (int i = 1; i < 256; i++) {
            count[i] += count[i - 1];
        }
        
        for (int i = n - 1; i >= 0; i--) {
            char ch = pos < strlen(strs[i]) ? strs[i][pos] : 0;
            output[--count[(unsigned char)ch]] = strs[i];
        }
        
        for (int i = 0; i < n; i++) {
            strs[i] = output[i];
        }
        
        free(output);
    }
}
```
