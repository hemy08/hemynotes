# 堆排序

## 概述

堆排序利用堆这种数据结构进行排序，通过构建大根堆，不断取出堆顶元素完成排序。

## 算法思想

1. 构建大根堆
2. 将堆顶元素与末尾元素交换
3. 对剩余元素重新堆化
4. 重复直到所有元素排序完成

## 算法演示

```
初始: 4  10  3  5  1

构建大根堆:
        10
       /  \
      5    3
     / \
    4   1

堆顶与末尾交换: 1  5  3  4 [10]
重新堆化:       5  4  3  1 [10]

继续:           4  1  3 [5  10]
继续:           3  1 [4  5  10]
继续:           1 [3  4  5  10]

结果:           1  3  4  5  10
```

## 基本实现

```c
void heapify(int arr[], int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;
    
    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }
    
    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }
    
    if (largest != i) {
        int temp = arr[i];
        arr[i] = arr[largest];
        arr[largest] = temp;
        
        heapify(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    
    for (int i = n - 1; i > 0; i--) {
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        
        heapify(arr, i, 0);
    }
}
```

## 迭代堆化

```c
void heapifyIterative(int arr[], int n, int i) {
    while (1) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;
        
        if (left < n && arr[left] > arr[largest]) {
            largest = left;
        }
        
        if (right < n && arr[right] > arr[largest]) {
            largest = right;
        }
        
        if (largest == i) break;
        
        int temp = arr[i];
        arr[i] = arr[largest];
        arr[largest] = temp;
        
        i = largest;
    }
}
```

## C++ 实现

```cpp
template<typename T>
void heapify(std::vector<T>& arr, int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;
    
    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;
    
    if (largest != i) {
        std::swap(arr[i], arr[largest]);
        heapify(arr, n, largest);
    }
}

template<typename T>
void heapSort(std::vector<T>& arr) {
    int n = arr.size();
    
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    
    for (int i = n - 1; i > 0; i--) {
        std::swap(arr[0], arr[i]);
        heapify(arr, i, 0);
    }
}
```

## 复杂度分析

| 操作 | 时间复杂度 |
|------|-----------|
| 建堆 | O(n) |
| 单次堆化 | O(log n) |
| 排序 | O(n log n) |

| 情况 | 时间复杂度 |
|------|-----------|
| 最好 | O(n log n) |
| 平均 | O(n log n) |
| 最坏 | O(n log n) |

空间复杂度：O(1)

## 稳定性

堆排序是**不稳定排序**：

```
原序列: 3A  3B  1
大根堆:    3A
          /  \
        3B    1

交换后: 1  3B [3A]  -> 顺序改变
```

## 建堆复杂度证明

```
高度h的节点数 ≤ n/2^(h+1)
高度h的节点堆化时间 = O(h)

总时间 = Σ(n/2^(h+1) * h) = O(n * Σ(h/2^(h+1)))
       = O(n * 1) = O(n)
```

## 堆排序应用

### 求第K大元素

```c
int findKthLargest(int arr[], int n, int k) {
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    
    for (int i = n - 1; i > n - k; i--) {
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        heapify(arr, i, 0);
    }
    
    return arr[0];
}
```

### Top K问题

```c
void topKSmallest(int arr[], int n, int k) {
    for (int i = k / 2 - 1; i >= 0; i--) {
        heapify(arr, k, i);
    }
    
    for (int i = k; i < n; i++) {
        if (arr[i] < arr[0]) {
            arr[0] = arr[i];
            heapify(arr, k, 0);
        }
    }
}
```

## 堆排序 vs 快速排序

| 特性 | 堆排序 | 快速排序 |
|------|--------|---------|
| 平均时间 | O(n log n) | O(n log n) |
| 最坏时间 | O(n log n) | O(n²) |
| 空间 | O(1) | O(log n) |
| 缓存 | 不友好 | 友好 |
| 稳定性 | 不稳定 | 不稳定 |

## 适用场景

1. **原地排序**：空间受限时
2. **大数据排序**：保证O(n log n)
3. **Top K问题**：维护大小为K的堆
4. **优先队列**：堆排序基础
