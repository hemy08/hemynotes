# 桶排序

## 概述

桶排序将数据分到多个桶中，每个桶内单独排序，然后合并所有桶。适用于均匀分布的数据。

## 算法思想

1. 创建n个空桶
2. 将每个元素放入对应的桶
3. 对每个桶内部排序
4. 合并所有桶的结果

## 算法演示

```
初始: 0.78  0.17  0.39  0.26  0.72  0.94  0.21  0.12  0.23  0.68

分桶（范围[0,1)，10个桶）:
桶0 [0.0-0.1): 
桶1 [0.1-0.2): 0.17  0.12
桶2 [0.2-0.3): 0.26  0.21  0.23
桶3 [0.3-0.4): 0.39
桶4 [0.4-0.5): 
桶5 [0.5-0.6): 
桶6 [0.6-0.7): 0.68
桶7 [0.7-0.8): 0.78  0.72
桶8 [0.8-0.9): 
桶9 [0.9-1.0): 0.94

桶内排序后合并:
0.12  0.17  0.21  0.23  0.26  0.39  0.68  0.72  0.78  0.94
```

## 基本实现

```c
#include <stdlib.h>

typedef struct Node {
    double value;
    struct Node *next;
} Node;

void bucketSort(double arr[], int n) {
    Node **buckets = (Node**)calloc(n, sizeof(Node*));
    
    for (int i = 0; i < n; i++) {
        int index = (int)(n * arr[i]);
        
        Node *newNode = (Node*)malloc(sizeof(Node));
        newNode->value = arr[i];
        newNode->next = buckets[index];
        buckets[index] = newNode;
    }
    
    for (int i = 0; i < n; i++) {
        Node *curr = buckets[i];
        while (curr) {
            Node *next = curr->next;
            while (next) {
                if (curr->value > next->value) {
                    double temp = curr->value;
                    curr->value = next->value;
                    next->value = temp;
                }
                next = next->next;
            }
            curr = curr->next;
        }
    }
    
    int index = 0;
    for (int i = 0; i < n; i++) {
        Node *curr = buckets[i];
        while (curr) {
            arr[index++] = curr->value;
            Node *temp = curr;
            curr = curr->next;
            free(temp);
        }
    }
    
    free(buckets);
}
```

## 整数桶排序

```c
void bucketSortInt(int arr[], int n, int maxVal) {
    int bucketCount = maxVal / 100 + 1;
    
    int **buckets = (int**)malloc(bucketCount * sizeof(int*));
    int *counts = (int*)calloc(bucketCount, sizeof(int));
    int *sizes = (int*)malloc(bucketCount * sizeof(int));
    
    for (int i = 0; i < bucketCount; i++) {
        sizes[i] = 100;
        buckets[i] = (int*)malloc(sizes[i] * sizeof(int));
    }
    
    for (int i = 0; i < n; i++) {
        int index = arr[i] / 100;
        if (counts[index] >= sizes[index]) {
            sizes[index] *= 2;
            buckets[index] = (int*)realloc(buckets[index], sizes[index] * sizeof(int));
        }
        buckets[index][counts[index]++] = arr[i];
    }
    
    for (int i = 0; i < bucketCount; i++) {
        if (counts[i] > 1) {
            qsort(buckets[i], counts[i], sizeof(int), 
                  (int(*)(const void*, const void*))strcmp);
        }
    }
    
    int index = 0;
    for (int i = 0; i < bucketCount; i++) {
        for (int j = 0; j < counts[i]; j++) {
            arr[index++] = buckets[i][j];
        }
        free(buckets[i]);
    }
    
    free(buckets);
    free(counts);
    free(sizes);
}
```

## C++ 实现

```cpp
void bucketSort(std::vector<double>& arr) {
    int n = arr.size();
    if (n <= 1) return;
    
    std::vector<std::vector<double>> buckets(n);
    
    for (double num : arr) {
        int index = (int)(n * num);
        if (index >= n) index = n - 1;
        buckets[index].push_back(num);
    }
    
    for (auto& bucket : buckets) {
        std::sort(bucket.begin(), bucket.end());
    }
    
    int index = 0;
    for (const auto& bucket : buckets) {
        for (double num : bucket) {
            arr[index++] = num;
        }
    }
}
```

## 复杂度分析

| 情况 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 平均 | O(n + k) | O(n + k) |
| 最坏 | O(n²) | O(n + k) |

k = 桶数量

## 稳定性

桶排序的稳定性取决于桶内排序算法。

## 桶数量选择

```
桶数量建议:
- 数据均匀分布: n个桶
- 数据范围已知: (max - min) / bucketSize
- 内存受限: 减少桶数量
```

## 应用场景

1. **浮点数排序**：[0,1)范围
2. **均匀分布数据**：效率最高
3. **大数据排序**：可并行处理
4. **外部排序**：每个桶写入文件

## 桶排序 vs 计数排序 vs 基数排序

| 特性 | 桶排序 | 计数排序 | 基数排序 |
|------|--------|---------|---------|
| 数据类型 | 任意 | 整数 | 整数 |
| 时间复杂度 | O(n+k)平均 | O(n+k) | O(d(n+k)) |
| 空间复杂度 | O(n+k) | O(k) | O(n+k) |
| 适用场景 | 均匀分布 | 范围小 | 固定进制 |

## 并行桶排序

```cpp
#include <thread>
#include <mutex>

void parallelBucketSort(std::vector<int>& arr, int numThreads) {
    int n = arr.size();
    int minVal = *std::min_element(arr.begin(), arr.end());
    int maxVal = *std::max_element(arr.begin(), arr.end());
    
    int range = maxVal - minVal + 1;
    int bucketSize = (range + numThreads - 1) / numThreads;
    
    std::vector<std::vector<int>> buckets(numThreads);
    std::mutex mtx;
    
    auto worker = [&](int start, int end) {
        for (int i = start; i < end; i++) {
            int index = (arr[i] - minVal) / bucketSize;
            if (index >= numThreads) index = numThreads - 1;
            
            std::lock_guard<std::mutex> lock(mtx);
            buckets[index].push_back(arr[i]);
        }
    };
    
    std::vector<std::thread> threads;
    int chunkSize = n / numThreads;
    
    for (int i = 0; i < numThreads; i++) {
        int start = i * chunkSize;
        int end = (i == numThreads - 1) ? n : start + chunkSize;
        threads.emplace_back(worker, start, end);
    }
    
    for (auto& t : threads) t.join();
    
    for (auto& bucket : buckets) {
        std::sort(bucket.begin(), bucket.end());
    }
    
    int index = 0;
    for (const auto& bucket : buckets) {
        for (int num : bucket) {
            arr[index++] = num;
        }
    }
}
```
