# 插入排序

## 概述

插入排序将数组分为已排序和未排序两部分，每次将未排序部分的第一个元素插入到已排序部分的适当位置。

## 算法思想

1. 将第一个元素视为已排序序列
2. 取下一个元素，在已排序序列中从后向前扫描
3. 找到适当位置插入
4. 重复直到所有元素排序完成

## 算法演示

```
初始: 5  3  8  4  2

[5] 3  8  4  2  -> [3  5] 8  4  2
[3  5] 8  4  2  -> [3  5  8] 4  2
[3  5  8] 4  2  -> [3  4  5  8] 2
[3  4  5  8] 2  -> [2  3  4  5  8]
```

## 基本实现

```c
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        
        arr[j + 1] = key;
    }
}
```

## 优化版本

### 二分插入排序

```c
int binarySearch(int arr[], int left, int right, int key) {
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == key) return mid + 1;
        if (arr[mid] < key) left = mid + 1;
        else right = mid - 1;
    }
    return left;
}

void binaryInsertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int pos = binarySearch(arr, 0, i - 1, key);
        
        for (int j = i - 1; j >= pos; j--) {
            arr[j + 1] = arr[j];
        }
        
        arr[pos] = key;
    }
}
```

### 希尔排序

```c
void shellSort(int arr[], int n) {
    for (int gap = n / 2; gap > 0; gap /= 2) {
        for (int i = gap; i < n; i++) {
            int key = arr[i];
            int j = i;
            
            while (j >= gap && arr[j - gap] > key) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            
            arr[j] = key;
        }
    }
}
```

## 复杂度分析

| 情况 | 时间复杂度 | 比较次数 | 移动次数 |
|------|-----------|---------|---------|
| 最好 | O(n) | n-1 | 0 |
| 平均 | O(n²) | n²/4 | n²/4 |
| 最坏 | O(n²) | n(n-1)/2 | n(n-1)/2 |

空间复杂度：O(1)

## 稳定性

插入排序是**稳定排序**，相等元素的相对顺序保持不变。

## 适用场景

1. **小规模数据**：n较时效率高
2. **基本有序数据**：移动次数少
3. **在线排序**：可以边接收边排序
4. **链表排序**：链表插入效率高

## 链表插入排序

```c
typedef struct ListNode {
    int val;
    struct ListNode *next;
} ListNode;

ListNode* insertionSortList(ListNode *head) {
    if (!head || !head->next) return head;
    
    ListNode dummy = {0, NULL};
    ListNode *curr = head;
    
    while (curr) {
        ListNode *next = curr->next;
        ListNode *prev = &dummy;
        
        while (prev->next && prev->next->val < curr->val) {
            prev = prev->next;
        }
        
        curr->next = prev->next;
        prev->next = curr;
        curr = next;
    }
    
    return dummy.next;
}
```
