# 计数排序

## 概述

计数排序是非比较排序算法，通过统计每个元素出现次数进行排序，适用于整数排序，时间复杂度O(n+k)。

## 算法思想

1. 统计每个元素出现次数
2. 计算元素累计次数（确定位置）
3. 根据计数将元素放到正确位置

## 算法演示

```
初始: 4  2  2  8  3  3  1

计数: 
值:   1  2  3  4  5  6  7  8
次数: 1  2  2  1  0  0  0  1

累计:
值:   1  2  3  4  5  6  7  8
位置: 1  3  5  6  6  6  6  7

结果: 1  2  2  3  3  4  8
```

## 基本实现

```c
void countingSort(int arr[], int n) {
    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
    }
    
    int *count = (int*)calloc(max + 1, sizeof(int));
    
    for (int i = 0; i < n; i++) {
        count[arr[i]]++;
    }
    
    int index = 0;
    for (int i = 0; i <= max; i++) {
        while (count[i] > 0) {
            arr[index++] = i;
            count[i]--;
        }
    }
    
    free(count);
}
```

## 稳定计数排序

```c
void countingSortStable(int arr[], int n) {
    int max = arr[0], min = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
        if (arr[i] < min) min = arr[i];
    }
    
    int range = max - min + 1;
    int *count = (int*)calloc(range, sizeof(int));
    int *output = (int*)malloc(n * sizeof(int));
    
    for (int i = 0; i < n; i++) {
        count[arr[i] - min]++;
    }
    
    for (int i = 1; i < range; i++) {
        count[i] += count[i - 1];
    }
    
    for (int i = n - 1; i >= 0; i--) {
        output[count[arr[i] - min] - 1] = arr[i];
        count[arr[i] - min]--;
    }
    
    for (int i = 0; i < n; i++) {
        arr[i] = output[i];
    }
    
    free(count);
    free(output);
}
```

## C++ 实现

```cpp
void countingSort(std::vector<int>& arr) {
    if (arr.empty()) return;
    
    int maxVal = *std::max_element(arr.begin(), arr.end());
    int minVal = *std::min_element(arr.begin(), arr.end());
    
    int range = maxVal - minVal + 1;
    std::vector<int> count(range, 0);
    
    for (int num : arr) {
        count[num - minVal]++;
    }
    
    int index = 0;
    for (int i = 0; i < range; i++) {
        while (count[i] > 0) {
            arr[index++] = i + minVal;
            count[i]--;
        }
    }
}
```

## 复杂度分析

| 情况 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 所有 | O(n + k) | O(k) |

k = max - min + 1，数据范围

## 稳定性

计数排序是**稳定排序**。

## 适用条件

1. **整数排序**：元素必须是整数
2. **范围有限**：k不能太大
3. **非负整数**：需要处理负数偏移

## 应用场景

1. **成绩排序**：分数范围有限
2. **年龄排序**：年龄范围有限
3. **基数排序基础**：按位排序
4. **字符统计**：ASCII字符排序

## 局限性

```
不适用情况:
- 数据范围远大于元素个数
- 浮点数排序
- 大整数排序
```
