# 排序算法概述

## 排序算法分类

排序算法可分为以下几类：

| 类别 | 算法 | 特点 |
|------|------|------|
| 插入排序 | 直接插入、希尔排序 | 适合小规模或基本有序数据 |
| 交换排序 | 冒泡排序、快速排序 | 通过交换元素位置排序 |
| 选择排序 | 直接选择、堆排序 | 每次选择最值 |
| 归并排序 | 归并排序 | 分治思想，稳定 |
| 基数排序 | 计数排序、基数排序、桶排序 | 非比较排序 |

## 排序算法比较

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 |
|------|---------|---------|------|--------|
| 冒泡排序 | O(n²) | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(n²) | O(1) | 稳定 |
| 希尔排序 | O(n^1.3) | O(n²) | O(1) | 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) | O(n²) | O(log n) | 不稳定 |
| 堆排序 | O(n log n) | O(n²) | O(1) | 不稳定 |
| 计数排序 | O(n+k) | O(n+k) | O(k) | 稳定 |
| 基数排序 | O(d(n+k)) | O(d(n+k)) | O(n+k) | 稳定 |

## 稳定性

**稳定排序**：相等元素的相对顺序保持不变

```
原序列: 3A, 1, 3B, 2
排序后: 1, 2, 3A, 3B (稳定)
        1, 2, 3B, 3A (不稳定)
```

## 内部排序 vs 外部排序

- **内部排序**：数据在内存中完成排序
- **外部排序**：数据量大，需要借助外存（归并排序）

## 排序下界

基于比较的排序算法，最坏情况下至少需要 Ω(n log n) 次比较。

## 常用排序函数

### C标准库

```c
#include <stdlib.h>

int compare(const void *a, const void *b) {
    return (*(int*)a - *(int*)b);
}

int arr[] = {3, 1, 4, 1, 5, 9, 2, 6};
qsort(arr, 8, sizeof(int), compare);
```

### C++ STL

```cpp
#include <algorithm>
#include <vector>

std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

std::sort(v.begin(), v.end());
std::stable_sort(v.begin(), v.end());
std::partial_sort(v.begin(), v.begin() + 3, v.end());
```

## 选择排序算法

| 数据规模 | 数据特点 | 推荐算法 |
|---------|---------|---------|
| n < 50 | - | 插入排序 |
| n中等 | 基本有序 | 插入排序 |
| n大 | - | 快速排序 |
| n大 | 稳定性要求 | 归并排序 |
| n大 | 空间受限 | 堆排序 |
| 数据范围小 | 整数 | 计数排序 |
