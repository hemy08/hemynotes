# 冒泡排序

## 概述

冒泡排序是最简单的排序算法，通过重复交换相邻的逆序元素，使最大元素逐渐"冒泡"到数组末尾。

## 算法思想

1. 比较相邻元素，如果逆序则交换
2. 每趟将最大元素移到末尾
3. 重复n-1趟完成排序

## 算法演示

```
初始: 5  3  8  4  2

第1趟: 3  5  4  2 [8]
第2趟: 3  4  2 [5  8]
第3趟: 3  2 [4  5  8]
第4趟: 2 [3  4  5  8]

结果: 2  3  4  5  8
```

## 基本实现

```c
void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
```

### 记录最后交换位置

```c
void bubbleSortBest(int arr[], int n) {
    int lastSwap = n - 1;
    while (lastSwap > 0) {
        int currSwap = 0;
        for (int j = 0; j < lastSwap; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                currSwap = j;
            }
        }
        lastSwap = currSwap;
    }
}
```

### 双向冒泡（鸡尾酒排序）

```c
void cocktailSort(int arr[], int n) {
    int left = 0, right = n - 1;
    
    while (left < right) {
        int swapped = 0;
        
        for (int j = left; j < right; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = 1;
            }
        }
        right--;
        
        for (int j = right; j > left; j--) {
            if (arr[j - 1] > arr[j]) {
                int temp = arr[j - 1];
                arr[j - 1] = arr[j];
                arr[j] = temp;
                swapped = 1;
            }
        }
        left++;
        
        if (!swapped) break;
    }
}
```

## C++ 实现

```cpp
template<typename T>
void bubbleSort(std::vector<T>& arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                std::swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        if (!swapped) break;
    }
}
```

## 复杂度分析

### 时间复杂度

| 情况 | 时间复杂度 | 说明 |
|------|-----------|------|
| 最好 | O(n) | 已有序，一趟检测 |
| 平均 | O(n²) | - |
| 最坏 | O(n²) | 逆序，完全交换 |

### 空间复杂度

O(1)，原地排序

### 比较次数

- 最多：n(n-1)/2
- 最少：n-1

### 交换次数

- 最多：n(n-1)/2
- 最少：0

## 稳定性分析

冒泡排序是**稳定排序**：

```
比较相邻元素时，只在 > 时交换
相等元素不会交换，相对顺序保持不变
```

## 适用场景

1. **教学演示**：算法简单易懂
2. **小规模数据**：n较小时效率可接受
3. **基本有序数据**：优化后效率较高

## 不适用场景

1. **大规模数据**：O(n²)效率太低
2. **性能要求高**：应选择快速排序等

## 变体应用

### 求第K大元素

```c
int findKthLargest(int arr[], int n, int k) {
    for (int i = 0; i < k; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr[n - k];
}
```

### 检测有序性

```c
int isSorted(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        if (arr[i] > arr[i + 1]) {
            return 0;
        }
    }
    return 1;
}
```
