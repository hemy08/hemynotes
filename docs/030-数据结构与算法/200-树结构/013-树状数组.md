# 树形数组（树状数组）

## 概述

树状数组（Binary Indexed Tree，BIT）是一种支持单点修改和区间查询的数据结构，代码简洁，效率高。

## 树状数组结构

```
数组: [1, 3, 5, 7, 9, 11, 13, 15]

树状数组:
C[1] = A[1] = 1
C[2] = A[1] + A[2] = 4
C[3] = A[3] = 5
C[4] = A[1] + A[2] + A[3] + A[4] = 16
C[5] = A[5] = 9
C[6] = A[5] + A[6] = 20
C[7] = A[7] = 13
C[8] = A[1] + ... + A[8] = 64
```

## 核心操作

### Lowbit函数

```c
int lowbit(int x) {
    return x & (-x);
}
```

### 单点更新

```c
void update(int tree[], int n, int index, int value) {
    while (index <= n) {
        tree[index] += value;
        index += lowbit(index);
    }
}
```

### 前缀查询

```c
int query(int tree[], int index) {
    int sum = 0;
    while (index > 0) {
        sum += tree[index];
        index -= lowbit(index);
    }
    return sum;
}
```

### 区间查询

```c
int rangeQuery(int tree[], int l, int r) {
    return query(tree, r) - query(tree, l - 1);
}
```

## 完整实现

```c
#include <stdlib.h>
#include <string.h>

typedef struct {
    int *tree;
    int *arr;
    int n;
} BIT;

BIT* createBIT(int n) {
    BIT *bit = (BIT*)malloc(sizeof(BIT));
    bit->n = n;
    bit->tree = (int*)calloc(n + 1, sizeof(int));
    bit->arr = (int*)calloc(n + 1, sizeof(int));
    return bit;
}

void initBIT(BIT *bit, int arr[], int n) {
    for (int i = 1; i <= n; i++) {
        bit->arr[i] = arr[i - 1];
        update(bit->tree, n, i, arr[i - 1]);
    }
}

void updateBIT(BIT *bit, int index, int value) {
    int delta = value - bit->arr[index];
    bit->arr[index] = value;
    update(bit->tree, bit->n, index, delta);
}

int queryBIT(BIT *bit, int index) {
    return query(bit->tree, index);
}

int rangeQueryBIT(BIT *bit, int l, int r) {
    return rangeQuery(bit->tree, l, r);
}

void destroyBIT(BIT *bit) {
    free(bit->tree);
    free(bit->arr);
    free(bit);
}
```

## 区间更新+单点查询

```c
typedef struct {
    int *diff;
    int n;
} BITDiff;

BITDiff* createBITDiff(int n) {
    BITDiff *bit = (BITDiff*)malloc(sizeof(BITDiff));
    bit->n = n;
    bit->diff = (int*)calloc(n + 1, sizeof(int));
    return bit;
}

void rangeUpdateDiff(BITDiff *bit, int l, int r, int value) {
    int index = l;
    while (index <= bit->n) {
        bit->diff[index] += value;
        index += lowbit(index);
    }
    
    index = r + 1;
    while (index <= bit->n) {
        bit->diff[index] -= value;
        index += lowbit(index);
    }
}

int pointQueryDiff(BITDiff *bit, int index) {
    int sum = 0;
    while (index > 0) {
        sum += bit->diff[index];
        index -= lowbit(index);
    }
    return sum;
}
```

## C++ 实现

```cpp
class BIT {
private:
    std::vector<int> tree;
    int n;
    
public:
    BIT(int size) : n(size), tree(size + 1, 0) {}
    
    void update(int index, int value) {
        while (index <= n) {
            tree[index] += value;
            index += index & (-index);
        }
    }
    
    int query(int index) {
        int sum = 0;
        while (index > 0) {
            sum += tree[index];
            index -= index & (-index);
        }
        return sum;
    }
    
    int rangeQuery(int l, int r) {
        return query(r) - query(l - 1);
    }
};
```

## 应用：逆序对计数

```c
long long countInversions(int nums[], int n) {
    int maxVal = 0;
    for (int i = 0; i < n; i++) {
        if (nums[i] > maxVal) maxVal = nums[i];
    }
    
    int *tree = (int*)calloc(maxVal + 1, sizeof(int));
    long long count = 0;
    
    for (int i = n - 1; i >= 0; i--) {
        int smaller = query(tree, nums[i] - 1);
        count += smaller;
        update(tree, maxVal, nums[i], 1);
    }
    
    free(tree);
    return count;
}
```

## 应用：动态求第K小

```c
int findKth(int tree[], int n, int k) {
    int pos = 0;
    int bitMask = 1;
    
    while (bitMask <= n) bitMask <<= 1;
    bitMask >>= 1;
    
    for (; bitMask; bitMask >>= 1) {
        int next = pos + bitMask;
        if (next <= n && tree[next] < k) {
            k -= tree[next];
            pos = next;
        }
    }
    
    return pos + 1;
}
```

## 树状数组 vs 线段树

| 特性 | 树状数组 | 线段树 |
|------|---------|--------|
| 代码复杂度 | 简单 | 复杂 |
| 空间 | O(n) | O(4n) |
| 单点修改 | O(log n) | O(log n) |
| 区间查询 | O(log n) | O(log n) |
| 区间修改 | 需要变体 | 原生支持 |

## 时间复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 建树 | O(n log n) |
| 单点更新 | O(log n) |
| 前缀查询 | O(log n) |
| 区间查询 | O(log n) |

## 应用场景

1. **区间求和**：动态维护前缀和
2. **逆序对**：计数排序思想
3. **动态排名**：实时统计
4. **频率统计**：元素计数
