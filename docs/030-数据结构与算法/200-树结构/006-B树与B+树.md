# B树与B+树

## 概述

B树和B+树是多路平衡查找树，专为磁盘存储设计，广泛应用于数据库索引和文件系统。

## B树特点

1. **多路分支**：每个节点最多m个孩子，m称为阶数
2. **节点关键字**：非根节点至少⌈m/2⌉-1个关键字，最多m-1个
3. **叶子同层**：所有叶子节点在同一层
4. **有序存储**：节点内关键字有序
5. **平衡性**：所有节点子树高度相同

## B树节点结构

```
阶数m = 4的B树节点:

     | K1 | K2 | K3 |
    /    |    |    \
  P0    P1   P2    P3

关键字: K1 < K2 < K3
子树: P0中的所有key < K1
      K1 < P1中的所有key < K2
      K2 < P2中的所有key < K3
      K3 < P3中的所有key
```

## B树示例

```
阶数m = 4的B树:

                | 30 | 60 |
               /     |     \
        |10|20|   |40|50|   |70|80|
```

## B树复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 查找 | O(log n) |
| 插入 | O(log n) |
| 删除 | O(log n) |
| 磁盘I/O | O(log_m n) |

## B树实现

### 节点定义

```c
#define M 4

typedef struct BTreeNode {
    int keys[M - 1];
    struct BTreeNode *children[M];
    int keyCount;
    int isLeaf;
} BTreeNode;

BTreeNode* createNode(int isLeaf) {
    BTreeNode *node = (BTreeNode*)malloc(sizeof(BTreeNode));
    node->keyCount = 0;
    node->isLeaf = isLeaf;
    for (int i = 0; i < M; i++) {
        node->children[i] = NULL;
    }
    return node;
}
```

### 查找操作

```c
int findKey(BTreeNode *node, int key) {
    int idx = 0;
    while (idx < node->keyCount && node->keys[idx] < key) {
        idx++;
    }
    return idx;
}

BTreeNode* search(BTreeNode *root, int key, int *pos) {
    if (root == NULL) return NULL;
    
    int idx = findKey(root, key);
    
    if (idx < root->keyCount && root->keys[idx] == key) {
        *pos = idx;
        return root;
    }
    
    if (root->isLeaf) return NULL;
    
    return search(root->children[idx], key, pos);
}
```

### 分裂节点

```c
void splitChild(BTreeNode *parent, int index) {
    BTreeNode *fullChild = parent->children[index];
    BTreeNode *newNode = createNode(fullChild->isLeaf);
    
    int mid = M / 2;
    newNode->keyCount = mid - 1;
    
    for (int i = 0; i < mid - 1; i++) {
        newNode->keys[i] = fullChild->keys[mid + 1 + i];
    }
    
    if (!fullChild->isLeaf) {
        for (int i = 0; i < mid; i++) {
            newNode->children[i] = fullChild->children[mid + 1 + i];
        }
    }
    
    fullChild->keyCount = mid - 1;
    
    for (int i = parent->keyCount; i > index; i--) {
        parent->children[i + 1] = parent->children[i];
    }
    parent->children[index + 1] = newNode;
    
    for (int i = parent->keyCount - 1; i >= index; i--) {
        parent->keys[i + 1] = parent->keys[i];
    }
    parent->keys[index] = fullChild->keys[mid];
    parent->keyCount++;
}
```

### 插入操作

```c
void insertNonFull(BTreeNode *node, int key) {
    int i = node->keyCount - 1;
    
    if (node->isLeaf) {
        while (i >= 0 && node->keys[i] > key) {
            node->keys[i + 1] = node->keys[i];
            i--;
        }
        node->keys[i + 1] = key;
        node->keyCount++;
    } else {
        while (i >= 0 && node->keys[i] > key) {
            i--;
        }
        i++;
        
        if (node->children[i]->keyCount == M - 1) {
            splitChild(node, i);
            if (node->keys[i] < key) {
                i++;
            }
        }
        insertNonFull(node->children[i], key);
    }
}

BTreeNode* insert(BTreeNode *root, int key) {
    if (root == NULL) {
        root = createNode(1);
        root->keys[0] = key;
        root->keyCount = 1;
        return root;
    }
    
    if (root->keyCount == M - 1) {
        BTreeNode *newRoot = createNode(0);
        newRoot->children[0] = root;
        splitChild(newRoot, 0);
        insertNonFull(newRoot, key);
        return newRoot;
    }
    
    insertNonFull(root, key);
    return root;
}
```

### 遍历操作

```c
void traverse(BTreeNode *root) {
    if (root == NULL) return;
    
    int i;
    for (i = 0; i < root->keyCount; i++) {
        if (!root->isLeaf) {
            traverse(root->children[i]);
        }
        printf("%d ", root->keys[i]);
    }
    
    if (!root->isLeaf) {
        traverse(root->children[i]);
    }
}
```

## B+树特点

1. **数据存储**：所有数据存储在叶子节点
2. **内部节点**：只存储索引，不存数据
3. **叶子链表**：叶子节点通过指针连接成链表
4. **节点大小**：内部节点和叶子节点关键字数量相同
5. **范围查询**：叶子链表支持高效范围查询

## B+树结构

```
内部节点(索引):
        | 20 | 40 |
       /    |     \
      v     v      v

叶子节点(数据):
|10|20| <-> |30|40| <-> |50|60|70|
```

## B+树实现

### 节点定义

```c
typedef struct BPlusTreeNode {
    int keys[M];
    struct BPlusTreeNode *children[M + 1];
    struct BPlusTreeNode *next;
    int keyCount;
    int isLeaf;
} BPlusTreeNode;

BPlusTreeNode* createBPlusNode(int isLeaf) {
    BPlusTreeNode *node = (BPlusTreeNode*)malloc(sizeof(BPlusTreeNode));
    node->keyCount = 0;
    node->isLeaf = isLeaf;
    node->next = NULL;
    for (int i = 0; i <= M; i++) {
        node->children[i] = NULL;
    }
    return node;
}
```

### 范围查询

```c
void rangeQuery(BPlusTreeNode *root, int low, int high) {
    BPlusTreeNode *leaf = root;
    
    while (!leaf->isLeaf) {
        int i = 0;
        while (i < leaf->keyCount && leaf->keys[i] <= low) {
            i++;
        }
        leaf = leaf->children[i];
    }
    
    while (leaf != NULL) {
        for (int i = 0; i < leaf->keyCount; i++) {
            if (leaf->keys[i] >= low && leaf->keys[i] <= high) {
                printf("%d ", leaf->keys[i]);
            } else if (leaf->keys[i] > high) {
                return;
            }
        }
        leaf = leaf->next;
    }
}
```

## B树 vs B+树

| 特性 | B树 | B+树 |
|------|-----|------|
| 数据存储 | 所有节点 | 仅叶子节点 |
| 查找路径 | 可能提前终止 | 必须到叶子 |
| 范围查询 | 需要中序遍历 | 叶子链表遍历 |
| 节点利用率 | 较低 | 较高 |
| 空间开销 | 较大 | 较小 |
| 适用场景 | 随机查找 | 范围查询 |

## 数据库索引应用

### MySQL InnoDB

```
B+树索引结构:
- 主键索引(聚簇索引): 叶子存储完整行数据
- 辅助索引(二级索引): 叶子存储主键值
- 联合索引: 多列组合,遵循最左前缀原则
```

### 索引查询

```sql
CREATE INDEX idx_name ON table_name(column);

SELECT * FROM table_name WHERE column BETWEEN 10 AND 100;
```

## 磁盘I/O分析

### B树磁盘I/O

```
树高度 h = log_m(n)
每次查找磁盘I/O次数 = h

示例:
n = 100万记录
m = 100 (节点大小约4KB)
h = log_100(1000000) ≈ 3次I/O
```

### B+树优势

```
- 叶子节点链表: 范围查询只需遍历叶子
- 更大分支因子: 内部节点不存数据,可存更多索引
- 更低树高: 减少磁盘I/O
```

## 性能优化

### 节点大小选择

```
节点大小 ≈ 磁盘块大小 (通常4KB-8KB)
目的: 一次I/O读取整个节点
```

### 缓冲区管理

```c
typedef struct {
    BTreeNode *node;
    int isDirty;
    time_t lastAccess;

BufferSlot buffer[BUFFER_SIZE];

BTreeNode* readNode(int nodeId) {
    if (inBuffer(nodeId)) {

    }

}
```
