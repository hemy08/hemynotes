# 贪心算法

## 概述

贪心算法在每一步选择当前最优解，希望通过局部最优选择达到全局最优。适用于具有贪心选择性质的问题。

## 贪心要素

1. **贪心选择性质**：局部最优选择能导向全局最优
2. **最优子结构**：问题的最优解包含子问题的最优解

## 经典问题

### 活动选择

```c
typedef struct {
    int start;
    int end;
} Activity;

int compare(const void *a, const void *b) {
    return ((Activity*)a)->end - ((Activity*)b)->end;
}

void activitySelection(Activity activities[], int n) {
    qsort(activities, n, sizeof(Activity), compare);
    
    printf("Selected activities: 0 ");
    int lastEnd = activities[0].end;
    
    for (int i = 1; i < n; i++) {
        if (activities[i].start >= lastEnd) {
            printf("%d ", i);
            lastEnd = activities[i].end;
        }
    }
    printf("\n");
}
```

### 分数背包

```c
typedef struct {
    int weight;
    int value;
    double ratio;
} Item;

int compare(const void *a, const void *b) {
    return ((Item*)b)->ratio > ((Item*)a)->ratio ? 1 : -1;
}

double fractionalKnapsack(Item items[], int n, int capacity) {
    for (int i = 0; i < n; i++) {
        items[i].ratio = (double)items[i].value / items[i].weight;
    }
    
    qsort(items, n, sizeof(Item), compare);
    
    double totalValue = 0;
    int remaining = capacity;
    
    for (int i = 0; i < n && remaining > 0; i++) {
        if (items[i].weight <= remaining) {
            totalValue += items[i].value;
            remaining -= items[i].weight;
        } else {
            totalValue += items[i].ratio * remaining;
            remaining = 0;
        }
    }
    
    return totalValue;
}
```

### 霍夫曼编码

```c
typedef struct HuffmanNode {
    char data;
    int freq;
    struct HuffmanNode *left, *right;
} HuffmanNode;

HuffmanNode* createNode(char data, int freq) {
    HuffmanNode *node = (HuffmanNode*)malloc(sizeof(HuffmanNode));
    node->data = data;
    node->freq = freq;
    node->left = node->right = NULL;
    return node;
}

void printCodes(HuffmanNode *root, char code[], int top) {
    if (root->left) {
        code[top] = '0';
        printCodes(root->left, code, top + 1);
    }
    if (root->right) {
        code[top] = '1';
        printCodes(root->right, code, top + 1);
    }
    if (!root->left && !root->right) {
        code[top] = '\0';
        printf("%c: %s\n", root->data, code);
    }
}
```

### 最小生成树（Prim）

```c
#define V 5

int minKey(int key[], int mstSet[]) {
    int min = INT_MAX, minIndex;
    
    for (int v = 0; v < V; v++) {
        if (!mstSet[v] && key[v] < min) {
            min = key[v];
            minIndex = v;
        }
    }
    
    return minIndex;
}

void primMST(int graph[V][V]) {
    int parent[V];
    int key[V];
    int mstSet[V];
    
    for (int i = 0; i < V; i++) {
        key[i] = INT_MAX;
        mstSet[i] = 0;
    }
    
    key[0] = 0;
    parent[0] = -1;
    
    for (int count = 0; count < V - 1; count++) {
        int u = minKey(key, mstSet);
        mstSet[u] = 1;
        
        for (int v = 0; v < V; v++) {
            if (graph[u][v] && !mstSet[v] && graph[u][v] < key[v]) {
                parent[v] = u;
                key[v] = graph[u][v];
            }
        }
    }
}
```

### 最小生成树（Kruskal）

```c
typedef struct {
    int src, dest, weight;
} Edge;

int parent[V];

int find(int i) {
    while (parent[i] != i) {
        i = parent[i];
    }
    return i;
}

void unionSets(int i, int j) {
    int a = find(i);
    int b = find(j);
    parent[a] = b;
}

void kruskalMST(Edge edges[], int E) {
    for (int i = 0; i < V; i++) parent[i] = i;
    
    Edge result[V - 1];
    int e = 0, i = 0;
    
    while (e < V - 1 && i < E) {
        Edge next = edges[i++];
        
        int x = find(next.src);
        int y = find(next.dest);
        
        if (x != y) {
            result[e++] = next;
            unionSets(x, y);
        }
    }
}
```

### 单源最短路径（Dijkstra）

```c
void dijkstra(int graph[V][V], int src) {
    int dist[V];
    int sptSet[V];
    
    for (int i = 0; i < V; i++) {
        dist[i] = INT_MAX;
        sptSet[i] = 0;
    }
    
    dist[src] = 0;
    
    for (int count = 0; count < V - 1; count++) {
        int min = INT_MAX, u;
        
        for (int v = 0; v < V; v++) {
            if (!sptSet[v] && dist[v] <= min) {
                min = dist[v];
                u = v;
            }
        }
        
        sptSet[u] = 1;
        
        for (int v = 0; v < V; v++) {
            if (!sptSet[v] && graph[u][v] && 
                dist[u] != INT_MAX && 
                dist[u] + graph[u][v] < dist[v]) {
                dist[v] = dist[u] + graph[u][v];
            }
        }
    }
}
```

### 跳跃游戏

```c
int canJump(int nums[], int n) {
    int maxReach = 0;
    
    for (int i = 0; i < n; i++) {
        if (i > maxReach) return 0;
        if (maxReach >= n - 1) return 1;
        
        int reach = i + nums[i];
        if (reach > maxReach) maxReach = reach;
    }
    
    return 1;
}
```

### 加油站问题

```c
int canCompleteCircuit(int gas[], int cost[], int n) {
    int total = 0, tank = 0, start = 0;
    
    for (int i = 0; i < n; i++) {
        total += gas[i] - cost[i];
        tank += gas[i] - cost[i];
        
        if (tank < 0) {
            start = i + 1;
            tank = 0;
        }
    }
    
    return total >= 0 ? start : -1;
}
```

### 区间调度

```c
int eraseOverlapIntervals(int intervals[][2], int n) {
    if (n <= 1) return 0;
    
    int count = 1;
    int end = intervals[0][1];
    
    for (int i = 1; i < n; i++) {
        if (intervals[i][0] >= end) {
            count++;
            end = intervals[i][1];
        }
    }
    
    return n - count;
}
```

## 贪心证明方法

### 交换论证

证明贪心选择与其他最优解可以交换而不影响最优性。

### 钱币找零证明

```
面额: 1, 5, 10, 25 (美分)
证明每次选择最大面额正确:

1. 若用两个10分替换一个25分，需增加硬币
2. 若用三个5分替换一个10分，需增加硬币
3. 贪心选择最优
```

## 贪心 vs 动态规划

| 特性 | 贪心 | 动态规划 |
|------|------|---------|
| 选择方式 | 每步最优 | 考虑所有可能 |
| 时间复杂度 | 通常O(n log n) | 通常O(n²)或更高 |
| 空间复杂度 | O(1)或O(n) | 通常O(n²) |
| 适用条件 | 贪心选择性质 | 最优子结构 |

## 贪心失败案例

### 0-1背包问题

```
物品: (价值, 重量) = (60, 10), (100, 20), (120, 30)
容量: 50

贪心(价值密度): 选择(60,10), (100,20), 剩余20不够放(120,30)
贪心结果: 160

最优解: 选择(100,20), (120,30) = 220
贪心失败!
```

## 应用场景

1. **调度问题**：活动选择、任务调度
2. **图算法**：最短路径、最小生成树
3. **编码压缩**：霍夫曼编码
4. **区间问题**：区间调度、区间覆盖
