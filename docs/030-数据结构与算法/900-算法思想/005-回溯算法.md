# 回溯算法

## 概述

回溯算法通过试探所有可能的解，当发现当前路径无法得到有效解时回溯，尝试其他路径。本质是深度优先搜索。

## 回溯框架

```c
void backtrack(参数) {
    if (终止条件) {
        保存结果;
        return;
    }
    
    for (选择 : 选择列表) {
        做选择;
        backtrack(新参数);
        撤销选择;
    }
}
```

## 经典问题

### 全排列

```c
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

void permute(int nums[], int n, int index) {
    if (index == n) {
        for (int i = 0; i < n; i++) printf("%d ", nums[i]);
        printf("\n");
        return;
    }
    
    for (int i = index; i < n; i++) {
        swap(&nums[index], &nums[i]);
        permute(nums, n, index + 1);
        swap(&nums[index], &nums[i]);
    }
}
```

### N皇后

```c
int isSafe(int board[], int row, int col) {
    for (int i = 0; i < row; i++) {
        if (board[i] == col) return 0;
        if (abs(board[i] - col) == abs(i - row)) return 0;
    }
    return 1;
}

void solveNQueens(int board[], int n, int row) {
    if (row == n) {
        for (int i = 0; i < n; i++) printf("%d ", board[i]);
        printf("\n");
        return;
    }
    
    for (int col = 0; col < n; col++) {
        if (isSafe(board, row, col)) {
            board[row] = col;
            solveNQueens(board, n, row + 1);
        }
    }
}
```

### 数独求解

```c
int isValid(int board[9][9], int row, int col, int num) {
    for (int i = 0; i < 9; i++) {
        if (board[row][i] == num) return 0;
        if (board[i][col] == num) return 0;
    }
    
    int boxRow = row / 3 * 3;
    int boxCol = col / 3 * 3;
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            if (board[boxRow + i][boxCol + j] == num) return 0;
        }
    }
    
    return 1;
}

int solveSudoku(int board[9][9]) {
    for (int row = 0; row < 9; row++) {
        for (int col = 0; col < 9; col++) {
            if (board[row][col] == 0) {
                for (int num = 1; num <= 9; num++) {
                    if (isValid(board, row, col, num)) {
                        board[row][col] = num;
                        if (solveSudoku(board)) return 1;
                        board[row][col] = 0;
                    }
                }
                return 0;
            }
        }
    }
    return 1;
}
```

### 括号生成

```c
void generateParenthesis(int n, int open, int close, char result[], int index) {
    if (index == 2 * n) {
        result[index] = '\0';
        printf("%s\n", result);
        return;
    }
    
    if (open < n) {
        result[index] = '(';
        generateParenthesis(n, open + 1, close, result, index + 1);
    }
    
    if (close < open) {
        result[index] = ')';
        generateParenthesis(n, open, close + 1, result, index + 1);
    }
}
```

### 组合总和

```c
void combinationSum(int candidates[], int n, int target, 
                    int result[], int resultSize, int start) {
    if (target == 0) {
        for (int i = 0; i < resultSize; i++) printf("%d ", result[i]);
        printf("\n");
        return;
    }
    
    for (int i = start; i < n; i++) {
        if (candidates[i] > target) continue;
        
        result[resultSize] = candidates[i];
        combinationSum(candidates, n, target - candidates[i], 
                       result, resultSize + 1, i);
    }
}
```

### 分割回文串

```c
int isPalindrome(char *s, int left, int right) {
    while (left < right) {
        if (s[left] != s[right]) return 0;
        left++;
        right--;
    }
    return 1;
}

void partition(char *s, int n, int start, char result[][100], int resultSize) {
    if (start == n) {
        for (int i = 0; i < resultSize; i++) printf("%s ", result[i]);
        printf("\n");
        return;
    }
    
    for (int end = start; end < n; end++) {
        if (isPalindrome(s, start, end)) {
            int len = end - start + 1;
            strncpy(result[resultSize], s + start, len);
            result[resultSize][len] = '\0';
            partition(s, n, end + 1, result, resultSize + 1);
        }
    }
}
```

### 单词搜索

```c
int exist(char board[][10], int m, int n, char *word, 
          int row, int col, int index, int visited[][10]) {
    if (word[index] == '\0') return 1;
    if (row < 0 || row >= m || col < 0 || col >= n) return 0;
    if (visited[row][col] || board[row][col] != word[index]) return 0;
    
    visited[row][col] = 1;
    
    int found = exist(board, m, n, word, row + 1, col, index + 1, visited) ||
                exist(board, m, n, word, row - 1, col, index + 1, visited) ||
                exist(board, m, n, word, row, col + 1, index + 1, visited) ||
                exist(board, m, n, word, row, col - 1, index + 1, visited);
    
    visited[row][col] = 0;
    
    return found;
}
```

## 剪枝优化

### 排序剪枝

```c
void combinationSum2(int candidates[], int n, int target, 
                     int result[], int resultSize, int start) {
    if (target == 0) {
        for (int i = 0; i < resultSize; i++) printf("%d ", result[i]);
        printf("\n");
        return;
    }
    
    for (int i = start; i < n; i++) {
        if (i > start && candidates[i] == candidates[i - 1]) continue;
        if (candidates[i] > target) break;
        
        result[resultSize] = candidates[i];
        combinationSum2(candidates, n, target - candidates[i], 
                        result, resultSize + 1, i + 1);
    }
}
```

## 回溯复杂度

| 问题 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 全排列 | O(n * n!) | O(n) |
| 组合 | O(C(n,k) * k) | O(k) |
| 子集 | O(n * 2^n) | O(n) |
| N皇后 | O(N!) | O(N) |

## 回溯 vs 深度优先搜索

| 特性 | 回溯 | DFS |
|------|------|-----|
| 目的 | 找所有解/最优解 | 遍历所有状态 |
| 状态恢复 | 需要撤销选择 | 不需要 |
| 剪枝 | 常用 | 较少使用 |
| 应用 | 组合优化问题 | 图遍历 |

## 应用场景

1. **排列组合**：全排列、组合、子集
2. **约束满足**：N皇后、数独、图着色
3. **路径搜索**：迷宫、单词搜索
4. **分割问题**：字符串分割、分割回文串
