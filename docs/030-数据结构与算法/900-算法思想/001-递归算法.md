# 递归算法

## 概述

递归是函数直接或间接调用自身的编程技术，通过将问题分解为相同性质的子问题求解。

## 递归要素

1. **基本情况**：直接返回结果的终止条件
2. **递归情况**：问题分解为更小的子问题
3. **递归关系**：原问题与子问题的关系

## 递归示例

### 阶乘

```c
long long factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```

### 斐波那契数列

```c
long long fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

long long fibonacciOptimized(int n) {
    if (n <= 1) return n;
    
    long long a = 0, b = 1;
    for (int i = 2; i <= n; i++) {
        long long temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}
```

### 汉诺塔

```c
void hanoi(int n, char from, char to, char aux) {
    if (n == 1) {
        printf("Move disk 1 from %c to %c\n", from, to);
        return;
    }
    hanoi(n - 1, from, aux, to);
    printf("Move disk %d from %c to %c\n", n, from, to);
    hanoi(n - 1, aux, to, from);
}
```

### 二分查找

```c
int binarySearch(int arr[], int left, int right, int target) {
    if (left > right) return -1;
    
    int mid = left + (right - left) / 2;
    
    if (arr[mid] == target) return mid;
    if (arr[mid] > target) return binarySearch(arr, left, mid - 1, target);
    return binarySearch(arr, mid + 1, right, target);
}
```

### 数组求和

```c
int sum(int arr[], int n) {
    if (n <= 0) return 0;
    return arr[n - 1] + sum(arr, n - 1);
}
```

### 字符串反转

```c
void reverseString(char str[], int left, int right) {
    if (left >= right) return;
    
    char temp = str[left];
    str[left] = str[right];
    str[right] = temp;
    
    reverseString(str, left + 1, right - 1);
}
```

### 全排列

```c
void swap(char *a, char *b) {
    char temp = *a;
    *a = *b;
    *b = temp;
}

void permute(char str[], int left, int right) {
    if (left == right) {
        printf("%s\n", str);
        return;
    }
    
    for (int i = left; i <= right; i++) {
        swap(&str[left], &str[i]);
        permute(str, left + 1, right);
        swap(&str[left], &str[i]);
    }
}
```

### 组合

```c
void combine(int arr[], int n, int r, int result[], int index, int start) {
    if (index == r) {
        for (int i = 0; i < r; i++) {
            printf("%d ", result[i]);
        }
        printf("\n");
        return;
    }
    
    for (int i = start; i <= n - r + index; i++) {
        result[index] = arr[i];
        combine(arr, n, r, result, index + 1, i + 1);
    }
}
```

## 递归转递推

### 使用数组缓存

```c
long long fib[1000];

long long fibonacciMemo(int n) {
    if (n <= 1) return n;
    if (fib[n] != 0) return fib[n];
    return fib[n] = fibonacciMemo(n - 1) + fibonacciMemo(n - 2);
}
```

### 使用尾递归

```c
long long factorialTail(int n, long long accumulator) {
    if (n <= 1) return accumulator;
    return factorialTail(n - 1, n * accumulator);
}

long long factorial(int n) {
    return factorialTail(n, 1);
}
```

## 递归复杂度分析

### 递推关系

| 递推关系 | 时间复杂度 | 示例 |
|---------|-----------|------|
| T(n) = T(n-1) + O(1) | O(n) | 阶乘 |
| T(n) = 2T(n-1) + O(1) | O(2^n) | 汉诺塔 |
| T(n) = T(n/2) + O(1) | O(log n) | 二分查找 |
| T(n) = 2T(n/2) + O(n) | O(n log n) | 归并排序 |
| T(n) = T(n-1) + T(n-2) | O(1.618^n) | 斐波那契 |

### 主定理

对于递推式 T(n) = aT(n/b) + f(n)：

```
若 f(n) = O(n^(log_b(a) - ε))，则 T(n) = Θ(n^log_b(a))
若 f(n) = Θ(n^log_b(a))，则 T(n) = Θ(n^log_b(a) * log n)
若 f(n) = Ω(n^(log_b(a) + ε))，则 T(n) = Θ(f(n))
```

## 递归问题

### 子集生成

```c
void subsets(int arr[], int n, int index, int selected[]) {
    if (index == n) {
        printf("{ ");
        for (int i = 0; i < n; i++) {
            if (selected[i]) printf("%d ", arr[i]);
        }
        printf("}\n");
        return;
    }
    
    selected[index] = 0;
    subsets(arr, n, index + 1, selected);
    
    selected[index] = 1;
    subsets(arr, n, index + 1, selected);
}
```

### 括号生成

```c
void generateParenthesis(int n, int open, int close, char result[], int index) {
    if (index == 2 * n) {
        result[index] = '\0';
        printf("%s\n", result);
        return;
    }
    
    if (open < n) {
        result[index] = '(';
        generateParenthesis(n, open + 1, close, result, index + 1);
    }
    
    if (close < open) {
        result[index] = ')';
        generateParenthesis(n, open, close + 1, result, index + 1);
    }
}
```

### 二叉树路径

```c
void printPaths(TreeNode *root, int path[], int pathLen) {
    if (root == NULL) return;
    
    path[pathLen] = root->data;
    pathLen++;
    
    if (root->left == NULL && root->right == NULL) {
        for (int i = 0; i < pathLen; i++) {
            printf("%d ", path[i]);
        }
        printf("\n");
    } else {
        printPaths(root->left, path, pathLen);
        printPaths(root->right, path, pathLen);
    }
}
```

## 递归优化

### 记忆化搜索

```c
#define MAX 1000
int memo[MAX];

int fibMemo(int n) {
    if (n <= 1) return n;
    if (memo[n] != -1) return memo[n];
    return memo[n] = fibMemo(n - 1) + fibMemo(n - 2);
}
```

### 剪枝优化

```c
int canSum(int target, int nums[], int n, int memo[]) {
    if (target == 0) return 1;
    if (target < 0) return 0;
    if (memo[target] != -1) return memo[target];
    
    for (int i = 0; i < n; i++) {
        if (canSum(target - nums[i], nums, n, memo)) {
            memo[target] = 1;
            return 1;
        }
    }
    
    memo[target] = 0;
    return 0;
}
```

## 递归栈溢出

### 问题

```c
void recurse(int n) {
    if (n == 0) return;
    recurse(n - 1);
}

recurse(1000000);  // 可能栈溢出
```

### 解决方案

```c
void recurseIterative(int n) {
    while (n > 0) {
        n--;
    }
}
```
