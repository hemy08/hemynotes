# 随机算法

## 概述

随机算法使用随机数作为输入的一部分，可用于提高算法效率或解决确定性算法难以解决的问题。

## 随机算法分类

| 类型 | 特点 | 示例 |
|------|------|------|
| Las Vegas | 总是正确，运行时间随机 | 随机快排 |
| Monte Carlo | 可能出错，运行时间确定 | 素性测试 |
| Sherwood | 总是正确，改善最坏情况 | 随机选择 |

## 随机快速排序

```c
#include <stdlib.h>
#include <time.h>

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int randomPartition(int arr[], int low, int high) {
    int random = low + rand() % (high - low + 1);
    swap(&arr[random], &arr[high]);
    
    int pivot = arr[high];
    int i = low - 1;
    
    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            swap(&arr[++i], &arr[j]);
        }
    }
    
    swap(&arr[i + 1], &arr[high]);
    return i + 1;
}

void randomQuickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = randomPartition(arr, low, high);
        randomQuickSort(arr, low, pi - 1);
        randomQuickSort(arr, pi + 1, high);
    }
}
```

## 随机选择（第K小元素）

```c
int randomSelect(int arr[], int low, int high, int k) {
    if (low == high) return arr[low];
    
    int pi = randomPartition(arr, low, high);
    int rank = pi - low + 1;
    
    if (k == rank) {
        return arr[pi];
    } else if (k < rank) {
        return randomSelect(arr, low, pi - 1, k);
    } else {
        return randomSelect(arr, pi + 1, high, k - rank);
    }
}
```

## 蓄水池抽样

```c
void reservoirSample(int stream[], int n, int k, int result[]) {
    for (int i = 0; i < k; i++) {
        result[i] = stream[i];
    }
    
    for (int i = k; i < n; i++) {
        int j = rand() % (i + 1);
        
        if (j < k) {
            result[j] = stream[i];
        }
    }
}
```

## 洗牌算法（Fisher-Yates）

```c
void shuffle(int arr[], int n) {
    for (int i = n - 1; i > 0; i--) {
        int j = rand() % (i + 1);
        swap(&arr[i], &arr[j]);
    }
}
```

## Miller-Rabin素性测试

```c
long long power(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    
    while (exp > 0) {
        if (exp & 1) {
            result = result * base % mod;
        }
        base = base * base % mod;
        exp >>= 1;
    }
    
    return result;
}

int millerRabin(long long n, int k) {
    if (n <= 1 || n == 4) return 0;
    if (n <= 3) return 1;
    
    long long d = n - 1;
    while (d % 2 == 0) {
        d /= 2;
    }
    
    for (int i = 0; i < k; i++) {
        long long a = 2 + rand() % (n - 4);
        long long x = power(a, d, n);
        
        if (x == 1 || x == n - 1) continue;
        
        while (d != n - 1) {
            x = x * x % n;
            d *= 2;
            
            if (x == 1) return 0;
            if (x == n - 1) break;
        }
        
        if (x != n - 1) return 0;
    }
    
    return 1;
}
```

## 随机游走

```c
int randomWalk(int graph[][100], int n, int start, int target, int maxSteps) {
    int current = start;
    int steps = 0;
    
    while (current != target && steps < maxSteps) {
        int degree = 0;
        int neighbors[100];
        
        for (int i = 0; i < n; i++) {
            if (graph[current][i]) {
                neighbors[degree++] = i;
            }
        }
        
        if (degree == 0) return -1;
        
        current = neighbors[rand() % degree];
        steps++;
    }
    
    return (current == target) ? steps : -1;
}
```

## 哈希函数（随机化）

```c
typedef struct {
    int a;
    int b;
    int p;
    int m;
} HashFunction;

HashFunction createRandomHash(int m, int p) {
    HashFunction hf;
    hf.a = 1 + rand() % (p - 1);
    hf.b = rand() % p;
    hf.p = p;
    hf.m = m;
    return hf;
}

int hashValue(HashFunction hf, int key) {
    return ((hf.a * key + hf.b) % hf.p) % hf.m;
}
```

## 蒙特卡洛积分

```c
double monteCarloIntegrate(double (*f)(double), double a, double b, int n) {
    double sum = 0;
    
    for (int i = 0; i < n; i++) {
        double x = a + (double)rand() / RAND_MAX * (b - a);
        sum += f(x);
    }
    
    return (b - a) * sum / n;
}
```

## C++ 随机数

```cpp
#include <random>
#include <algorithm>

std::mt19937 rng(std::random_device{}());

int randomInt(int min, int max) {
    std::uniform_int_distribution<int> dist(min, max);
    return dist(rng);
}

double randomDouble(double min, double max) {
    std::uniform_real_distribution<double> dist(min, max);
    return dist(rng);
}

template<typename T>
void shuffleVector(std::vector<T>& v) {
    std::shuffle(v.begin(), v.end(), rng);
}
```

## 时间复杂度

| 算法 | 期望时间 | 最坏时间 |
|------|---------|---------|
| 随机快排 | O(n log n) | O(n²) |
| 随机选择 | O(n) | O(n²) |
| Miller-Rabin | O(k log³ n) | O(k log³ n) |

## 应用场景

1. **排序算法**：避免最坏情况
2. **素性测试**：大数判断
3. **随机采样**：大数据抽样
4. **蒙特卡洛**：数值计算
5. **密码学**：随机密钥生成
