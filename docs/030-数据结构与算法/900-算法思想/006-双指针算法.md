# 双指针算法

## 概述

双指针算法使用两个指针协同遍历数据结构，常用于数组、链表等线性结构，可将O(n²)优化为O(n)。

## 双指针类型

| 类型 | 描述 | 应用 |
|------|------|------|
| 相向双指针 | 两端向中间移动 | 两数之和、反转数组 |
| 同向双指针 | 同方向移动 | 滑动窗口、快慢指针 |
| 快慢指针 | 不同速度移动 | 环检测、找中点 |

## 相向双指针

### 两数之和（有序数组）

```c
void twoSum(int arr[], int n, int target) {
    int left = 0, right = n - 1;
    
    while (left < right) {
        int sum = arr[left] + arr[right];
        
        if (sum == target) {
            printf("%d + %d = %d\n", arr[left], arr[right], target);
            return;
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    
    printf("No solution\n");
}
```

### 三数之和

```c
void threeSum(int arr[], int n, int target) {
    for (int i = 0; i < n - 2; i++) {
        int left = i + 1, right = n - 1;
        
        while (left < right) {
            int sum = arr[i] + arr[left] + arr[right];
            
            if (sum == target) {
                printf("%d + %d + %d = %d\n", 
                       arr[i], arr[left], arr[right], target);
                return;
            } else if (sum < target) {
                left++;
            } else {
                right--;
            }
        }
    }
}
```

### 反转数组

```c
void reverse(int arr[], int n) {
    int left = 0, right = n - 1;
    
    while (left < right) {
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        left++;
        right--;
    }
}
```

### 回文判断

```c
int isPalindrome(char *s, int n) {
    int left = 0, right = n - 1;
    
    while (left < right) {
        if (s[left] != s[right]) {
            return 0;
        }
        left++;
        right--;
    }
    
    return 1;
}
```

### 盛最多水的容器

```c
int maxArea(int height[], int n) {
    int left = 0, right = n - 1;
    int maxArea = 0;
    
    while (left < right) {
        int area = (right - left) * 
                   (height[left] < height[right] ? height[left] : height[right]);
        
        if (area > maxArea) maxArea = area;
        
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }
    
    return maxArea;
}
```

## 同向双指针

### 移除元素

```c
int removeElement(int arr[], int n, int val) {
    int slow = 0;
    
    for (int fast = 0; fast < n; fast++) {
        if (arr[fast] != val) {
            arr[slow++] = arr[fast];
        }
    }
    
    return slow;
}
```

### 删除有序数组重复元素

```c
int removeDuplicates(int arr[], int n) {
    if (n <= 1) return n;
    
    int slow = 1;
    
    for (int fast = 1; fast < n; fast++) {
        if (arr[fast] != arr[fast - 1]) {
            arr[slow++] = arr[fast];
        }
    }
    
    return slow;
}
```

### 移除有序数组重复元素（保留K个）

```c
int removeDuplicatesK(int arr[], int n, int k) {
    if (n <= k) return n;
    
    int slow = k;
    
    for (int fast = k; fast < n; fast++) {
        if (arr[fast] != arr[slow - k]) {
            arr[slow++] = arr[fast];
        }
    }
    
    return slow;
}
```

## 快慢指针

### 链表环检测

```c
typedef struct ListNode {
    int val;
    struct ListNode *next;
} ListNode;

int hasCycle(ListNode *head) {
    ListNode *slow = head;
    ListNode *fast = head;
    
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        
        if (slow == fast) {
            return 1;
        }
    }
    
    return 0;
}
```

### 找环入口

```c
ListNode* detectCycle(ListNode *head) {
    ListNode *slow = head;
    ListNode *fast = head;
    
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        
        if (slow == fast) {
            ListNode *ptr = head;
            while (ptr != slow) {
                ptr = ptr->next;
                slow = slow->next;
            }
            return ptr;
        }
    }
    
    return NULL;
}
```

### 链表中点

```c
ListNode* findMiddle(ListNode *head) {
    ListNode *slow = head;
    ListNode *fast = head;
    
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    
    return slow;
}
```

### 链表倒数第K个节点

```c
ListNode* getKthFromEnd(ListNode *head, int k) {
    ListNode *fast = head;
    ListNode *slow = head;
    
    for (int i = 0; i < k; i++) {
        if (fast) fast = fast->next;
    }
    
    while (fast) {
        fast = fast->next;
        slow = slow->next;
    }
    
    return slow;
}
```

## 滑动窗口

### 最大连续子数组和（固定长度）

```c
int maxSumFixed(int arr[], int n, int k) {
    int sum = 0;
    
    for (int i = 0; i < k; i++) {
        sum += arr[i];
    }
    
    int maxSum = sum;
    
    for (int i = k; i < n; i++) {
        sum += arr[i] - arr[i - k];
        if (sum > maxSum) maxSum = sum;
    }
    
    return maxSum;
}
```

### 最小覆盖子串

```c
char* minWindow(char *s, char *t) {
    int count[128] = {0};
    int required = 0;
    
    for (int i = 0; t[i]; i++) {
        if (count[t[i]] == 0) required++;
        count[t[i]]++;
    }
    
    int left = 0, right = 0;
    int formed = 0;
    int windowCounts[128] = {0};
    
    int minLen = 1e9, start = 0;
    
    while (s[right]) {
        windowCounts[s[right]]++;
        
        if (windowCounts[s[right]] == count[s[right]]) {
            formed++;
        }
        
        while (formed == required) {
            if (right - left + 1 < minLen) {
                minLen = right - left + 1;
                start = left;
            }
            
            windowCounts[s[left]]--;
            if (windowCounts[s[left]] < count[s[left]]) {
                formed--;
            }
            left++;
        }
        
        right++;
    }
    
    if (minLen == 1e9) return "";
    
    char *result = (char*)malloc(minLen + 1);
    strncpy(result, s + start, minLen);
    result[minLen] = '\0';
    return result;
}
```

## C++ 实现

```cpp
int twoSumClosest(std::vector<int>& arr, int target) {
    int left = 0, right = arr.size() - 1;
    int closest = INT_MAX;
    
    while (left < right) {
        int sum = arr[left] + arr[right];
        
        if (abs(sum - target) < abs(closest - target)) {
            closest = sum;
        }
        
        if (sum < target) left++;
        else right--;
    }
    
    return closest;
}
```

## 应用场景

1. **有序数组**：两数之和、三数之和
2. **链表问题**：环检测、找中点
3. **字符串问题**：回文、子串
4. **区间问题**：滑动窗口
