# 分治算法

## 概述

分治算法将问题分解为若干规模较小的相同问题，递归求解子问题，然后合并子问题的解得到原问题的解。

## 分治步骤

1. **分解**：将原问题分解为若干子问题
2. **解决**：递归求解子问题
3. **合并**：将子问题的解合并为原问题的解

## 经典分治算法

### 归并排序

```c
void merge(int arr[], int left, int mid, int right, int temp[]) {
    int i = left, j = mid + 1, k = left;
    
    while (i <= mid && j <= right) {
        temp[k++] = (arr[i] <= arr[j]) ? arr[i++] : arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    
    for (i = left; i <= right; i++) arr[i] = temp[i];
}

void mergeSort(int arr[], int left, int right, int temp[]) {
    if (left >= right) return;
    
    int mid = left + (right - left) / 2;
    mergeSort(arr, left, mid, temp);
    mergeSort(arr, mid + 1, right, temp);
    merge(arr, left, mid, right, temp);
}
```

### 快速排序

```c
int partition(int arr[], int low, int high) {
    int pivot = arr[high];
    int i = low - 1;
    
    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;
        }
    }
    
    int temp = arr[i + 1]; arr[i + 1] = arr[high]; arr[high] = temp;
    return i + 1;
}

void quickSort(int arr[], int low, int high) {
    if (low >= high) return;
    
    int pi = partition(arr, low, high);
    quickSort(arr, low, pi - 1);
    quickSort(arr, pi + 1, high);
}
```

### 二分查找

```c
int binarySearch(int arr[], int left, int right, int target) {
    if (left > right) return -1;
    
    int mid = left + (right - left) / 2;
    
    if (arr[mid] == target) return mid;
    if (arr[mid] > target) return binarySearch(arr, left, mid - 1, target);
    return binarySearch(arr, mid + 1, right, target);
}
```

### 最大子数组和

```c
int maxCrossingSum(int arr[], int left, int mid, int right) {
    int leftSum = INT_MIN, rightSum = INT_MIN;
    int sum = 0;
    
    for (int i = mid; i >= left; i--) {
        sum += arr[i];
        if (sum > leftSum) leftSum = sum;
    }
    
    sum = 0;
    for (int i = mid + 1; i <= right; i++) {
        sum += arr[i];
        if (sum > rightSum) rightSum = sum;
    }
    
    return leftSum + rightSum;
}

int maxSubArray(int arr[], int left, int right) {
    if (left == right) return arr[left];
    
    int mid = left + (right - left) / 2;
    
    int leftMax = maxSubArray(arr, left, mid);
    int rightMax = maxSubArray(arr, mid + 1, right);
    int crossMax = maxCrossingSum(arr, left, mid, right);
    
    int max = leftMax;
    if (rightMax > max) max = rightMax;
    if (crossMax > max) max = crossMax;
    
    return max;
}
```

### 矩阵乘法（Strassen）

```c
void matrixAdd(int n, int A[][n], int B[][n], int C[][n]) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            C[i][j] = A[i][j] + B[i][j];
        }
    }
}

void matrixSub(int n, int A[][n], int B[][n], int C[][n]) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            C[i][j] = A[i][j] - B[i][j];
        }
    }
}
```

### 大整数乘法（Karatsuba）

```c
long long karatsuba(long long x, long long y) {
    if (x < 10 || y < 10) return x * y;
    
    int n = 0;
    long long temp = x > y ? x : y;
    while (temp > 0) {
        temp /= 10;
        n++;
    }
    n = (n + 1) / 2;
    
    long long pow = 1;
    for (int i = 0; i < n; i++) pow *= 10;
    
    long long a = x / pow, b = x % pow;
    long long c = y / pow, d = y % pow;
    
    long long ac = karatsuba(a, c);
    long long bd = karatsuba(b, d);
    long long adbc = karatsuba(a + b, c + d) - ac - bd;
    
    return ac * pow * pow + adbc * pow + bd;
}
```

### 最近点对

```c
typedef struct {
    double x, y;
} Point;

double distance(Point p1, Point p2) {
    return sqrt((p1.x - p2.x) * (p1.x - p2.x) + 
                (p1.y - p2.y) * (p1.y - p2.y));
}

int compareX(const void *a, const void *b) {
    return ((Point*)a)->x > ((Point*)b)->x ? 1 : -1;
}

double closestPair(Point points[], int n) {
    if (n <= 3) {
        double min = DBL_MAX;
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                double d = distance(points[i], points[j]);
                if (d < min) min = d;
            }
        }
        return min;
    }
    
    qsort(points, n, sizeof(Point), compareX);
    
    int mid = n / 2;
    Point midPoint = points[mid];
    
    double dl = closestPair(points, mid);
    double dr = closestPair(points + mid, n - mid);
    double d = dl < dr ? dl : dr;
    
    Point strip[n];
    int stripCount = 0;
    
    for (int i = 0; i < n; i++) {
        if (abs(points[i].x - midPoint.x) < d) {
            strip[stripCount++] = points[i];
        }
    }
    
    for (int i = 0; i < stripCount; i++) {
        for (int j = i + 1; j < stripCount && 
             (strip[j].y - strip[i].y) < d; j++) {
            double dist = distance(strip[i], strip[j]);
            if (dist < d) d = dist;
        }
    }
    
    return d;
}
```

## 主定理

对于递推式 T(n) = aT(n/b) + f(n)：

```
令 c = log_b(a)

情况1: 若 f(n) = O(n^(c-ε))，则 T(n) = Θ(n^c)
情况2: 若 f(n) = Θ(n^c)，则 T(n) = Θ(n^c * log n)
情况3: 若 f(n) = Ω(n^(c+ε))，则 T(n) = Θ(f(n))
```

### 应用示例

| 算法 | 递推式 | 复杂度 |
|------|--------|--------|
| 二分查找 | T(n) = T(n/2) + O(1) | O(log n) |
| 归并排序 | T(n) = 2T(n/2) + O(n) | O(n log n) |
| 快速排序(平均) | T(n) = 2T(n/2) + O(n) | O(n log n) |
| Strassen | T(n) = 7T(n/2) + O(n²) | O(n^2.81) |
| Karatsuba | T(n) = 3T(n/2) + O(n) | O(n^1.58) |

## 分治问题

### 数组第K大元素

```c
int findKthLargest(int arr[], int left, int right, int k) {
    if (left == right) return arr[left];
    
    int pivotIndex = partition(arr, left, right);
    int rank = right - pivotIndex + 1;
    
    if (rank == k) return arr[pivotIndex];
    if (rank > k) return findKthLargest(arr, pivotIndex + 1, right, k);
    return findKthLargest(arr, left, pivotIndex - 1, k - rank);
}
```

### 逆序对计数

```c
long long mergeCount(int arr[], int left, int mid, int right, int temp[]) {
    long long count = 0;
    int i = left, j = mid + 1, k = left;
    
    while (i <= mid && j <= right) {
        if (arr[i] <= arr[j]) {
            temp[k++] = arr[i++];
        } else {
            temp[k++] = arr[j++];
            count += mid - i + 1;
        }
    }
    
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    
    for (i = left; i <= right; i++) arr[i] = temp[i];
    
    return count;
}

long long countInversions(int arr[], int left, int right, int temp[]) {
    if (left >= right) return 0;
    
    int mid = left + (right - left) / 2;
    long long count = 0;
    count += countInversions(arr, left, mid, temp);
    count += countInversions(arr, mid + 1, right, temp);
    count += mergeCount(arr, left, mid, right, temp);
    
    return count;
}
```

## 分治优化

### 减少递归深度

```c
void quickSortOptimized(int arr[], int low, int high) {
    while (low < high) {
        int pi = partition(arr, low, high);
        
        if (pi - low < high - pi) {
            quickSortOptimized(arr, low, pi - 1);
            low = pi + 1;
        } else {
            quickSortOptimized(arr, pi + 1, high);
            high = pi - 1;
        }
    }
}
```

## 分治 vs 动态规划

| 特性 | 分治 | 动态规划 |
|------|------|---------|
| 子问题关系 | 独立 | 重叠 |
| 计算顺序 | 自顶向下 | 自底向上 |
| 存储 | 不需要 | 需要记忆化 |
| 适用问题 | 子问题独立 | 子问题重叠 |

## 应用场景

1. **排序算法**：归并排序、快速排序
2. **查找算法**：二分查找
3. **几何算法**：最近点对、凸包
4. **矩阵运算**：Strassen算法
5. **大数运算**：Karatsuba乘法
