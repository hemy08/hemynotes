# 单调栈

## 概述

单调栈是一种特殊的栈，栈内元素保持单调递增或单调递减，常用于求解下一个更大/更小元素问题。

## 单调栈类型

| 类型 | 栈内顺序 | 适用场景 |
|------|---------|---------|
| 单调递增栈 | 栈底到栈顶递增 | 求下一个更小元素 |
| 单调递减栈 | 栈底到栈顶递减 | 求下一个更大元素 |

## 基本实现

### 下一个更大元素

```c
int* nextGreaterElement(int nums[], int n, int *returnSize) {
    int *result = (int*)malloc(sizeof(int) * n);
    int *stack = (int*)malloc(sizeof(int) * n);
    int top = -1;
    
    *returnSize = n;
    
    for (int i = n - 1; i >= 0; i--) {
        while (top >= 0 && stack[top] <= nums[i]) {
            top--;
        }
        
        result[i] = (top >= 0) ? stack[top] : -1;
        stack[++top] = nums[i];
    }
    
    free(stack);
    return result;
}
```

### 每日温度

```c
int* dailyTemperatures(int temperatures[], int n, int *returnSize) {
    int *result = (int*)calloc(n, sizeof(int));
    int *stack = (int*)malloc(sizeof(int) * n);
    int top = -1;
    
    *returnSize = n;
    
    for (int i = 0; i < n; i++) {
        while (top >= 0 && temperatures[stack[top]] < temperatures[i]) {
            result[stack[top]] = i - stack[top];
            top--;
        }
        
        stack[++top] = i;
    }
    
    free(stack);
    return result;
}
```

## 柱状图中最大矩形

```c
int largestRectangleArea(int heights[], int n) {
    int *stack = (int*)malloc(sizeof(int) * (n + 1));
    int top = -1;
    int maxArea = 0;
    
    for (int i = 0; i <= n; i++) {
        int h = (i == n) ? 0 : heights[i];
        
        while (top >= 0 && heights[stack[top]] > h) {
            int height = heights[stack[top--]];
            int width = (top < 0) ? i : i - stack[top] - 1;
            int area = height * width;
            
            if (area > maxArea) maxArea = area;
        }
        
        stack[++top] = i;
    }
    
    free(stack);
    return maxArea;
}
```

## 接雨水

```c
int trap(int height[], int n) {
    int *stack = (int*)malloc(sizeof(int) * n);
    int top = -1;
    int water = 0;
    
    for (int i = 0; i < n; i++) {
        while (top >= 0 && height[stack[top]] < height[i]) {
            int bottom = height[stack[top--]];
            
            if (top < 0) break;
            
            int left = stack[top];
            int h = (height[left] < height[i]) ? height[left] : height[i];
            int w = i - left - 1;
            
            water += (h - bottom) * w;
        }
        
        stack[++top] = i;
    }
    
    free(stack);
    return water;
}
```

## 左右两侧更小元素

```c
void findLeftRightSmaller(int nums[], int n, int left[], int right[]) {
    int *stack = (int*)malloc(sizeof(int) * n);
    int top = -1;
    
    for (int i = 0; i < n; i++) left[i] = -1;
    for (int i = 0; i < n; i++) right[i] = n;
    
    for (int i = 0; i < n; i++) {
        while (top >= 0 && nums[stack[top]] > nums[i]) {
            right[stack[top--]] = i;
        }
        
        if (top >= 0) {
            left[i] = stack[top];
        }
        
        stack[++top] = i;
    }
    
    free(stack);
}
```

## C++ 实现

```cpp
#include <vector>
#include <stack>

std::vector<int> nextGreaterElement(const std::vector<int>& nums) {
    int n = nums.size();
    std::vector<int> result(n, -1);
    std::stack<int> st;
    
    for (int i = n - 1; i >= 0; i--) {
        while (!st.empty() && st.top() <= nums[i]) {
            st.pop();
        }
        
        if (!st.empty()) {
            result[i] = st.top();
        }
        
        st.push(nums[i]);
    }
    
    return result;
}

int largestRectangleArea(std::vector<int>& heights) {
    std::stack<int> st;
    int maxArea = 0;
    int n = heights.size();
    
    for (int i = 0; i <= n; i++) {
        int h = (i == n) ? 0 : heights[i];
        
        while (!st.empty() && heights[st.top()] > h) {
            int height = heights[st.top()];
            st.pop();
            int width = st.empty() ? i : i - st.top() - 1;
            maxArea = std::max(maxArea, height * width);
        }
        
        st.push(i);
    }
    
    return maxArea;
}
```

## 时间复杂度

每个元素最多入栈出栈一次，时间复杂度 O(n)

## 应用场景

1. **下一个更大/更小元素**：单调栈经典应用
2. **柱状图最大矩形**：面积计算
3. **接雨水**：水量计算
4. **股票价格跨度**：前小于当前的天数
