# 斐波那契查找

## 概述

斐波那契查找是二分查找的变种，使用黄金分割比例分割数组，只使用加减运算，适合硬件实现。

## 斐波那契数列

```c
long long fib[50];

void initFib() {
    fib[0] = 1;
    fib[1] = 1;
    for (int i = 2; i < 50; i++) {
        fib[i] = fib[i - 1] + fib[i - 2];
    }
}
```

## 算法思想

将数组分割为大小为F(k-1)和F(k-2)的两部分，其中F(k)是最小的大于等于n的斐波那契数。

```
分割比例: F(k-1) : F(k-2) ≈ 1 : 0.618 (黄金分割)
```

## 基本实现

```c
int fibonacciSearch(int arr[], int n, int target) {
    long long fib[50];
    fib[0] = 1;
    fib[1] = 1;
    int k = 1;
    while (fib[k] < n) {
        k++;
        fib[k] = fib[k - 1] + fib[k - 2];
    }
    
    int *temp = (int*)malloc(sizeof(int) * fib[k]);
    for (int i = 0; i < n; i++) {
        temp[i] = arr[i];
    }
    for (int i = n; i < fib[k]; i++) {
        temp[i] = arr[n - 1];
    }
    
    int low = 0;
    int high = n - 1;
    
    while (low <= high) {
        int mid = low + fib[k - 1] - 1;
        
        if (temp[mid] == target) {
            free(temp);
            return mid < n ? mid : n - 1;
        }
        
        if (temp[mid] > target) {
            high = mid - 1;
            k -= 1;
        } else {
            low = mid + 1;
            k -= 2;
        }
    }
    
    free(temp);
    return -1;
}
```

## 优化版本

```c
int fibonacciSearchOptimized(int arr[], int n, int target) {
    if (n == 0) return -1;
    if (n == 1) return arr[0] == target ? 0 : -1;
    
    int fib2 = 1;
    int fib1 = 1;
    int fib = fib2 + fib1;
    
    while (fib < n) {
        fib2 = fib1;
        fib1 = fib;
        fib = fib2 + fib1;
    }
    
    int offset = -1;
    
    while (fib > 1) {
        int i = (offset + fib2 < n - 1) ? offset + fib2 : n - 1;
        
        if (arr[i] < target) {
            fib = fib1;
            fib1 = fib2;
            fib2 = fib - fib1;
            offset = i;
        } else if (arr[i] > target) {
            fib = fib2;
            fib1 = fib1 - fib2;
            fib2 = fib - fib1;
        } else {
            return i;
        }
    }
    
    if (fib1 && arr[offset + 1] == target) {
        return offset + 1;
    }
    
    return -1;
}
```

## C++ 实现

```cpp
int fibonacciSearch(const std::vector<int>& arr, int target) {
    int n = arr.size();
    if (n == 0) return -1;
    
    int fib2 = 1, fib1 = 1, fib = 2;
    
    while (fib < n) {
        fib2 = fib1;
        fib1 = fib;
        fib = fib2 + fib1;
    }
    
    int offset = -1;
    
    while (fib > 1) {
        int i = std::min(offset + fib2, n - 1);
        
        if (arr[i] < target) {
            fib = fib1;
            fib1 = fib2;
            fib2 = fib - fib1;
            offset = i;
        } else if (arr[i] > target) {
            fib = fib2;
            fib1 = fib1 - fib2;
            fib2 = fib - fib1;
        } else {
            return i;
        }
    }
    
    if (fib1 && offset + 1 < n && arr[offset + 1] == target) {
        return offset + 1;
    }
    
    return -1;
}
```

## 黄金分割比例

```
φ = (1 + √5) / 2 ≈ 1.618

斐波那契性质:
- F(n) / F(n-1) → φ (n增大时)
- F(n-1) / F(n) → 1/φ ≈ 0.618

分割位置 = low + F(k-1) - 1 ≈ low + 0.618 * (high - low)
```

## 三种查找算法比较

| 特性 | 二分查找 | 插值查找 | 斐波那契查找 |
|------|---------|---------|-------------|
| 分割方式 | 中点 | 估计位置 | 黄金分割点 |
| 运算 | 除法 | 除法 | 仅加减 |
| 平均时间 | O(log n) | O(log log n) | O(log n) |
| 硬件友好 | 一般 | 一般 | 好 |

## 应用场景

1. **硬件实现**：避免除法运算
2. **嵌入式系统**：资源受限环境
3. **均匀分布数据**：性能稳定
4. **有序数组查找**：通用场景
