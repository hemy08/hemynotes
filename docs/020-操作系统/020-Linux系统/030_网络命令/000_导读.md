# 网络命令

## 概述

Linux网络命令用于网络配置、诊断和监控。

## 主要命令

| 命令 | 说明 |
|------|------|
| curl | HTTP请求工具 |
| ip | 网络配置命令 |
| nslookup | DNS查询工具 |
| tcpdump | 网络抓包工具 |
| fuser | 查看端口占用进程 |
| lsof | 列出打开文件 |
| netstat | 网络连接统计 |
| ss | Socket统计 |
