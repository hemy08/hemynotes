# 文件操作命令

## 概述

Linux文件操作命令用于管理文件和目录，包括搜索、编辑、比较、复制、移动、删除等操作。

## 主要命令

| 命令 | 说明 |
|------|------|
| grep | 文本搜索工具 |
| rsync | 文件同步工具 |
| sed | 流编辑器 |
| sync | 文件系统同步 |
| diff | 文件比较工具 |
| cp | 文件复制命令 |
| setfacl | 设置文件访问控制列表 |
| find | 文件查找工具 |
| mv | 移动/重命名命令 |
| rm | 删除命令 |
