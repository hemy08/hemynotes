# 压缩与解压命令

## 概述

Linux压缩与解压命令用于创建和解压归档文件，支持多种压缩格式。

## 主要命令

| 命令 | 说明 | 扩展名 |
|------|------|--------|
| tar | 归档打包工具 | .tar, .tar.gz, .tar.bz2, .tar.xz |
| zip | ZIP压缩工具 | .zip |
| gzip | GNU压缩工具 | .gz |
| bzip2 | 高压缩率工具 | .bz2 |
| xz | LZMA压缩工具 | .xz |

## 压缩格式对比

| 格式 | 压缩率 | 速度 | 兼容性 |
|------|--------|------|--------|
| gzip | 中 | 快 | 最高 |
| bzip2 | 高 | 中 | 高 |
| xz | 最高 | 慢 | 中 |
| zip | 中 | 快 | 最高 |
