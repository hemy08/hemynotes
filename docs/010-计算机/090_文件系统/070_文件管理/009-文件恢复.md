# 文件恢复

## 概述

!!! note "文件恢复"
    恢复被删除或损坏的文件,尽可能找回丢失的数据。

## 恢复原理

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>恢复原理</strong>
    <p style="margin: 5px 0;">删除文件时,通常只删除目录项,数据仍保留在磁盘上。</p>
</div>

**关键点:**

- 数据块未被覆盖
- inode信息可能仍存在
- 及时恢复成功率高

## 恢复方法

### 从回收站恢复

!!! tip "回收站恢复"
    从系统回收站或废纸篓恢复文件。

**特点:**

- 最简单的方法
- 适合刚删除的文件
- Windows: 回收站
- Linux: 废纸篓
- macOS: 废纸篓

### 使用恢复工具

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>恢复工具</strong>
</div>

**Linux工具:**

- **extundelete**: EXT文件系统恢复
- **testdisk**: 分区恢复
- **photorec**: 数据恢复

**Windows工具:**

- **Recuva**: 免费恢复工具
- **EaseUS**: 数据恢复软件
- **R-Studio**: 专业恢复工具

### 从备份恢复

!!! info "备份恢复"
    从系统备份或个人备份中恢复文件。

**备份类型:**

- 系统备份
- 增量备份
- 云备份

## 恢复步骤

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>恢复步骤</strong>
</div>

1. 立即停止写入操作
2. 确定文件系统和存储设备
3. 选择合适的恢复工具
4. 扫描磁盘查找文件
5. 预览和选择文件
6. 恢复到安全位置

## 恢复注意事项

!!! warning "注意事项"
    - 立即停止使用磁盘
    - 不要安装新软件
    - 恢复到不同位置
    - 定期备份重要文件
    - 使用只读模式扫描

## 预防措施

<div style="background-color: #F3E5F5; padding: 15px; margin: 10px 0; border-left: 4px solid #9C27B0; border-radius: 5px;">
    <strong>预防措施</strong>
    <ul style="margin: 5px 0;">
        <li>定期备份重要文件</li>
        <li>使用版本控制系统</li>
        <li>启用文件系统日志</li>
        <li>使用RAID提高可靠性</li>
        <li>谨慎使用删除命令</li>
    </ul>
</div>
