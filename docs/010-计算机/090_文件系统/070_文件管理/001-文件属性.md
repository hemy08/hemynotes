# 文件属性

## 概述

!!! note "文件属性"
    文件属性是描述文件特征的信息,包括基本属性、存储属性、时间属性和权限属性。

## 基本属性

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>基本属性</strong>
    <ul style="margin: 5px 0;">
        <li><strong>文件名</strong>: 文件的符号名,用于标识文件</li>
        <li><strong>文件类型</strong>: 文件的格式和用途</li>
        <li><strong>文件大小</strong>: 文件的字节数</li>
        <li><strong>文件位置</strong>: 文件在存储设备上的位置</li>
    </ul>
</div>

## 存储属性

!!! tip "存储属性"
    描述文件在存储设备上的信息。

    - **起始块号**: 文件占用的第一个物理块
    - **块数量**: 文件占用的物理块数
    - **存储方式**: 连续、链接或索引
    - **索引指针**: 索引分配时的索引块位置

## 时间属性

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>时间属性</strong>
</div>

- **创建时间**: 文件创建的时间戳
- **修改时间**: 内容最后修改时间
- **访问时间**: 最后访问时间
- **状态改变时间**: 属性最后修改时间

## 权限属性

!!! info "权限属性"
    控制文件访问权限的信息。

    - **文件主**: 文件所有者
    - **所属组**: 文件所属用户组
    - **访问权限**: 读、写、执行权限
    - **特殊权限**: SUID、SGID、粘滞位

## 扩展属性

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>扩展属性</strong>
    <p style="margin: 5px 0;">用户自定义的附加属性。</p>
</div>

- 文件标签
- 文件注释
- 自定义元数据

## 属性查看命令

!!! warning "常用命令"
    - **ls -l**: 查看文件详细属性
    - **stat**: 查看文件完整属性
    - **file**: 查看文件类型
