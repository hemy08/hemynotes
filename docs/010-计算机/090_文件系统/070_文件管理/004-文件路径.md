# 文件路径

## 概述

!!! note "文件路径"
    文件路径是定位文件的字符串,描述文件在目录树中的位置。

## 路径类型

### 绝对路径

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>绝对路径</strong>
    <p style="margin: 5px 0;">从根目录开始的完整路径。</p>
</div>

**格式:**

- **Linux**: /home/user/documents/file.txt
- **Windows**: C:\Users\User\Documents\file.txt

**特点:**

- 路径唯一
- 不依赖当前目录
- 适合脚本和配置

### 相对路径

!!! tip "相对路径"
    从当前工作目录开始的路径。

**格式:**

- ./documents/file.txt: 当前目录下
- ../file.txt: 上级目录
- documents/file.txt: 当前目录下(省略./)

**特点:**

- 路径简短
- 依赖当前目录
- 适合临时操作

## 路径分隔符

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>分隔符</strong>
</div>

- **Linux/macOS**: / (正斜杠)
- **Windows**: \ (反斜杠)

**注意:** 现代系统通常都支持 /

## 特殊目录符号

!!! info "特殊符号"
    - **.**: 当前目录
    - **..**: 上级目录
    - **~**: 用户主目录(Linux/macOS)
    - **/**: 根目录

## 路径操作

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>路径操作</strong>
</div>

- **路径拼接**: 组合目录和文件名
- **路径解析**: 获取绝对路径
- **路径分解**: 提取目录和文件名
- **路径规范化**: 消除 . 和 ..
