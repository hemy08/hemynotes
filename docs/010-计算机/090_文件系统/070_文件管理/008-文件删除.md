# 文件删除

## 概述

!!! note "文件删除"
    从文件系统中移除文件或目录,释放存储空间。

## 删除类型

### 删除文件

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>删除文件</strong>
    <p style="margin: 5px 0;">删除单个或多个文件。</p>
</div>

**示例:**

`ash
rm file.txt
rm file1.txt file2.txt
`

### 删除目录

!!! tip "删除目录"
    删除空目录或递归删除目录。

**删除空目录:**

`ash
rmdir directory/
`

**递归删除:**

`ash
rm -r directory/
`

### 强制删除

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>强制删除</strong>
    <p style="margin: 5px 0;">不提示确认,直接删除。</p>
</div>

**示例:**

`ash
rm -rf directory/
`

## 删除选项

!!! info "常用选项"
    - **-r/-R**: 递归删除目录
    - **-f**: 强制删除
    - **-i**: 交互式确认
    - **-v**: 显示详细信息
    - **-d**: 删除空目录

## 删除过程

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>删除步骤</strong>
</div>

1. 检查文件是否存在
2. 检查删除权限
3. 如果是目录,检查是否为空
4. 删除目录项
5. 释放磁盘空间
6. 更新文件系统信息

## 删除原理

!!! warning "删除原理"
    删除操作通常只删除目录项,数据仍存在。

- **目录项删除**: 从目录中移除文件名
- **空间标记**: 标记磁盘块为空闲
- **数据保留**: 实际数据可能仍存在
- **可恢复性**: 删除后可能恢复

## 注意事项

<div style="background-color: #F3E5F5; padding: 15px; margin: 10px 0; border-left: 4px solid #9C27B0; border-radius: 5px;">
    <strong>注意事项</strong>
    <ul style="margin: 5px 0;">
        <li>删除操作不可逆</li>
        <li>谨慎使用 -rf 选项</li>
        <li>重要文件先备份</li>
        <li>检查删除权限</li>
    </ul>
</div>
