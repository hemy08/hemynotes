# 文件命名

## 概述

!!! note "文件命名"
    文件命名规则和约定,影响文件的可识别性和可管理性。

## 文件名组成

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>文件名组成</strong>
    <p style="margin: 5px 0;">文件名 = 主文件名 + 扩展名</p>
</div>

**示例:** document.txt

- 主文件名: document
- 扩展名: .txt

## 命名规则

### 长度限制

!!! tip "长度限制"
    不同系统对文件名长度有限制。

- **Windows**: 255字符(NTFS)
- **Linux**: 255字节
- **macOS**: 255字符

### 字符限制

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>允许字符</strong>
</div>

**Windows禁止字符:**

`
< > : " / \ | ? *
`

**Linux允许字符:**

除 / 和空字符外的所有字符

### 大小写敏感

!!! info "大小写敏感"
    - **Windows**: 不敏感(file.txt = FILE.TXT)
    - **Linux**: 敏感(file.txt ≠ FILE.TXT)
    - **macOS**: 默认不敏感

## 扩展名约定

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>常见扩展名</strong>
</div>

- **.txt**: 文本文件
- **.doc/.docx**: Word文档
- **.pdf**: PDF文档
- **.jpg/.png**: 图像文件
- **.mp3/.mp4**: 音视频文件
- **.exe**: Windows可执行文件
- **.sh**: Shell脚本

## 命名最佳实践

!!! warning "命名建议"
    - 使用有意义的名称
    - 避免特殊字符
    - 避免空格(用下划线或连字符)
    - 保持一致性
    - 避免过长的文件名
