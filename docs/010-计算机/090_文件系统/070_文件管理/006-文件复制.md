# 文件复制

## 概述

!!! note "文件复制"
    创建文件的副本,保留原文件不变。

## 复制类型

### 文件到文件

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>文件到文件</strong>
    <p style="margin: 5px 0;">复制单个文件到另一个文件。</p>
</div>

**示例:**

`ash
cp source.txt destination.txt
`

### 文件到目录

!!! tip "文件到目录"
    复制文件到目录中,保持原文件名。

**示例:**

`ash
cp file.txt /path/to/directory/
`

### 目录到目录

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>目录到目录</strong>
    <p style="margin: 5px 0;">递归复制整个目录。</p>
</div>

**示例:**

`ash
cp -r source_dir/ destination_dir/
`

## 复制选项

!!! info "常用选项"
    - **-r/-R**: 递归复制目录
    - **-p**: 保留文件属性
    - **-f**: 强制覆盖
    - **-i**: 交互式确认
    - **-v**: 显示详细信息
    - **-u**: 只复制更新的文件

## 复制过程

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>复制步骤</strong>
</div>

1. 检查源文件是否存在
2. 检查目标位置权限
3. 创建目标文件
4. 读取源文件数据
5. 写入目标文件
6. 复制文件属性

## 注意事项

!!! warning "注意事项"
    - 复制会覆盖同名文件
    - 需要足够的磁盘空间
    - 大文件复制耗时较长
    - 符号链接的处理
