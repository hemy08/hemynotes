# 文件类型

## 概述

!!! note "文件类型"
    根据文件的用途、内容和组织方式进行分类,便于管理和使用。

## 按用途分类

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>按用途分类</strong>
</div>

### 系统文件

!!! tip "系统文件"
    操作系统和系统程序使用的文件。

- 操作系统内核
- 系统配置文件
- 系统库文件

### 用户文件

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>用户文件</strong>
    <p style="margin: 5px 0;">用户创建和使用的文件。</p>
</div>

- 文档文件
- 程序文件
- 数据文件

### 库文件

!!! info "库文件"
    标准库和用户自定义库。

- 标准函数库
- 动态链接库(.so, .dll)
- 静态库(.a, .lib)

## 按内容分类

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>按内容分类</strong>
</div>

### 文本文件

- ASCII码文件
- UTF-8编码文件
- 配置文件

### 二进制文件

!!! warning "二进制文件"
    非文本格式的文件。

- 可执行文件
- 目标文件
- 数据库文件

### 媒体文件

<div style="background-color: #F3E5F5; padding: 15px; margin: 10px 0; border-left: 4px solid #9C27B0; border-radius: 5px;">
    <strong>媒体文件</strong>
</div>

- 图像文件: .jpg, .png, .gif
- 音频文件: .mp3, .wav, .flac
- 视频文件: .mp4, .avi, .mkv

## 按存取方式分类

!!! success "按存取方式"
    - **顺序文件**: 顺序存取
    - **随机文件**: 随机存取
    - **索引文件**: 索引存取
