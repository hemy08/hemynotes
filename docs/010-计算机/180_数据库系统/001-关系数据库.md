# 关系数据库

## 概述

!!! note "关系数据库"
    基于关系模型的数据库,使用二维表存储数据。

## 关系模型

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>关系模型组成</strong>
    <ul style="margin: 5px 0;">
        <li><strong>关系</strong>: 二维表</li>
        <li><strong>属性</strong>: 表的列</li>
        <li><strong>元组</strong>: 表的行</li>
        <li><strong>域</strong>: 属性的取值范围</li>
    </ul>
</div>

## 关系运算

### 传统集合运算

!!! tip "集合运算"
    - **并(∪)**: 两个关系的并集
    - **差(-)**: 两个关系的差集
    - **交(∩)**: 两个关系的交集
    - **笛卡尔积(×)**: 两个关系的组合

### 专门关系运算

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>关系运算</strong>
</div>

- **选择(σ)**: 选择满足条件的行
- **投影(π)**: 选择指定的列
- **连接(⋈)**: 连接两个关系
- **除(÷)**: 除运算

## 完整性约束

!!! info "完整性约束"
    - **实体完整性**: 主键不能为空
    - **参照完整性**: 外键约束
    - **用户定义完整性**: 用户自定义约束
