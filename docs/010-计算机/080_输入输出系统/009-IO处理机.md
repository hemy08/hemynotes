# I/O处理机

## 概述

!!! note "I/O处理机"
    I/O处理机是专门用于处理I/O操作的处理器,具有独立执行程序的能力。

## I/O处理机类型

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>I/O处理机分类</strong>
</div>

### 通道处理机

!!! tip "通道处理机"
    早期I/O处理机,功能相对简单。

**特点:**

- 执行通道程序
- 管理I/O控制器
- 与CPU共享内存

### 外围处理机

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>外围处理机(PPU)</strong>
    <p style="margin: 5px 0;">功能更强的I/O处理机。</p>
</div>

**特点:**

- 有自己的指令系统
- 有自己的内存
- 独立性强

## I/O处理机功能

!!! info "I/O处理机功能"
    - 执行I/O程序
    - 管理I/O设备
    - 处理I/O中断
    - 数据格式转换
    - 设备错误处理

## I/O处理机优点

!!! warning "I/O处理机优点"
    - CPU与I/O高度并行
    - 减轻CPU负担
    - 提高系统吞吐率
    - 适合大量I/O操作
