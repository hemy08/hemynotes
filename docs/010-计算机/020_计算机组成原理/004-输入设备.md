# 输入设备

## 概述

输入设备是向计算机输入数据和信息的设备,是计算机与用户或其他设备通信的桥梁。

!!! note "输入设备定义"
    输入设备(Input Device)是用户和计算机系统之间进行信息交换的主要装置之一,用于把原始数据和处理这些数据的程序输入到计算机中。

## 常见输入设备

### 1. 键盘

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>键盘(Keyboard)</strong>
    <p style="margin: 5px 0;">最基本、最常用的输入设备。</p>
</div>

**类型:**

- 机械键盘: 手感好,寿命长
- 薄膜键盘: 价格低,噪音小
- 无线键盘: 使用方便
- 人体工程学键盘: 减少疲劳

**工作原理:**

- 键盘扫描
- 键码识别
- 按键去抖
- 键码发送

### 2. 鼠标

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>鼠标(Mouse)</strong>
    <p style="margin: 5px 0;">图形用户界面下的主要输入设备。</p>
</div>

**类型:**

- 机械鼠标: 使用滚球
- 光电鼠标: 使用光学传感器
- 激光鼠标: 使用激光传感器
- 无线鼠标: 使用无线连接

**工作原理:**

- 位置检测
- 移动计算
- 按键检测
- 数据传输

### 3. 扫描仪

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>扫描仪(Scanner)</strong>
    <p style="margin: 5px 0;">将纸质文档转换为电子图像。</p>
</div>

**类型:**

- 平板式扫描仪
- 馈纸式扫描仪
- 手持式扫描仪
- 3D扫描仪

## 输入设备的工作原理

!!! tip "输入设备工作原理"
    输入设备通过以下步骤完成输入:

```mermaid
graph LR
    A[用户操作] --> B[信号转换]
    B --> C[数据编码]
    C --> D[接口传输]
    D --> E[计算机接收]
```

### 1. 信号转换

<div style="border: 2px solid #4CAF50; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>信号转换</strong>
    <p style="margin: 5px 0;">将用户的物理操作转换为电信号。</p>
</div>

### 2. 数据编码

<div style="border: 2px solid #2196F3; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>数据编码</strong>
    <p style="margin: 5px 0;">将电信号编码为计算机可识别的数据。</p>
</div>

### 3. 接口传输

<div style="border: 2px solid #FF9800; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>接口传输</strong>
    <p style="margin: 5px 0;">通过接口将数据传输到计算机。</p>
</div>

## 输入设备的接口

!!! info "常见接口类型"
    输入设备使用不同类型的接口连接计算机。

### 1. USB接口

- USB 2.0: 480Mbps
- USB 3.0: 5Gbps
- USB 3.1: 10Gbps
- USB Type-C: 正反插

### 2. PS/2接口

- 专用键盘接口
- 专用鼠标接口
- 不支持热插拔

### 3. 无线接口

- 蓝牙: 短距离无线
- 2.4G无线: 无线鼠标键盘
- 红外: 遥控器
