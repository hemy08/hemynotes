# CPU结构

## 概述

CPU(中央处理器)是计算机的核心部件,负责执行指令和处理数据。

!!! note "CPU定义"
    CPU(Central Processing Unit)是计算机的运算核心和控制核心,负责解释计算机指令和处理计算机软件中的数据。

## CPU的基本组成

```mermaid
graph TB
    A[CPU] --> B[运算器<br/>ALU]
    A --> C[控制器<br/>CU]
    A --> D[寄存器组]
    A --> E[内部总线]
    
    B --> B1[算术逻辑单元]
    B --> B2[累加器]
    B --> B3[状态寄存器]
    
    C --> C1[程序计数器]
    C --> C2[指令寄存器]
    C --> C3[指令译码器]
    C --> C4[时序发生器]
```

## 运算器

### 运算器的组成

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>运算器组成</strong>
    <ul style="margin: 5px 0;">
        <li>算术逻辑单元(ALU): 执行运算</li>
        <li>累加器(ACC): 存放操作数和结果</li>

- 算术运算: 加、减、乘、除
- 逻辑运算: 与、或、非、异或
- 移位操作: 左移、右移
- 比较操作: 等于、大于、小于

## 控制器

### 控制器的组成

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">

    <ul style="margin: 5px 0;">

### 控制器的功能

- 取指令
- 分析指令
- 执行指令
- 控制数据流向

### 常见寄存器

#### 1. 程序计数器(PC)

<div style="background-color: #FFF3E0; padding: 10px; margin: 10px 0; border-left: 4px solid #FF9800;">
    <strong>程序计数器(Program Counter)</strong>
    <p style="margin: 5px 0;">存放下一条指令的地址。</p>
</div>

#### 2. 指令寄存器(IR)

<div style="background-color: #F3E5F5; padding: 10px; margin: 10px 0; border-left: 4px solid #9C27B0;">
    <strong>指令寄存器(Instruction Register)</strong>
    <p style="margin: 5px 0;">存放当前正在执行的指令。</p>
</div>

#### 3. 状态寄存器(PSW)

<div style="background-color: #FCE4EC; padding: 10px; margin: 10px 0; border-left: 4px solid #E91E63;">
    <strong>状态寄存器(Program Status Word)</strong>
    <p style="margin: 5px 0;">存放运算结果的状态标志。</p>
</div>

**常见标志位:**

- ZF: 零标志(结果为零)
- SF: 符号标志(结果为负)
- CF: 进位标志(有进位)
- OF: 溢出标志(有溢出)

## CPU的工作过程

!!! info "CPU工作过程"
    CPU按以下循环工作:

```mermaid
graph LR
    A[取指令] --> B[分析指令]
    B --> C[执行指令]
    C --> A
```

### 1. 取指令

<div style="border: 2px solid #4CAF50; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>取指令(Fetch)</strong>
    <ol style="margin: 5px 0;">
        <li>PC → MAR (送指令地址)</li>
        <li>M(MAR) → MDR (读指令)</li>
        <li>MDR → IR (存指令)</li>
        <li>PC + 1 → PC (更新PC)</li>
    </ol>
</div>

### 2. 分析指令

<div style="border: 2px solid #2196F3; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>分析指令(Decode)</strong>
    <p style="margin: 5px 0;">对IR中的指令进行译码,识别指令类型。</p>
</div>

### 3. 执行指令

<div style="border: 2px solid #FF9800; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>执行指令(Execute)</strong>
    <p style="margin: 5px 0;">根据译码结果执行相应操作。</p>
</div>

## 多核CPU

!!! warning "多核CPU"
    在一个芯片上集成多个处理器核心。

**优点:**

- 提高并行处理能力
- 降低功耗
- 提高性能

**类型:**

- 同构多核: 所有核心相同
- 异构多核: 核心功能不同(如big.LITTLE)
