# 指令流水线

## 概述

!!! note "指令流水线"
    将指令执行过程分解为多个子过程,并行执行,提高CPU吞吐率。

## 流水线基本原理

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>流水线原理</strong>
    <p style="margin: 5px 0;">多条指令在不同阶段同时执行。</p>
</div>

## 流水线阶段

!!! tip "经典5级流水线"
    - **IF(取指)**: 从存储器取指令
    - **ID(译码)**: 指令译码,读寄存器
    - **EX(执行)**: 执行运算
    - **MEM(访存)**: 访问存储器
    - **WB(写回)**: 结果写回寄存器

## 流水线相关

!!! warning "流水线相关"
    引起流水线停顿的因素。

### 结构相关

- 多条指令同时使用同一资源
- 解决: 资源重复设置

### 数据相关

<div style="background-color: #F3E5F5; padding: 15px; margin: 10px 0; border-left: 4px solid #9C27B0; border-radius: 5px;">
    <strong>数据相关</strong>
    <p style="margin: 5px 0;">后续指令依赖前面指令的结果。</p>
</div>

- RAW(写后读)
- 解决: 数据前递、暂停流水线

### 控制相关

!!! success "控制相关"
    分支指令引起流水线断流。

- 解决: 分支预测、延迟槽
