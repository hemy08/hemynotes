# 寻址方式

## 概述

!!! note "寻址方式"
    寻址方式是指令中寻找操作数或操作数地址的方式。

## 常见寻址方式

### 1. 立即寻址

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>立即寻址</strong>
    <p style="margin: 5px 0;">操作数直接包含在指令中。</p>
</div>

**格式:** OP #n

**示例:** MOV AX, #100

**特点:** 速度快,无需访问内存

### 2. 直接寻址

!!! tip "直接寻址"
    指令中直接给出操作数的内存地址。

**格式:** OP (ADDR)

**示例:** MOV AX, (1000H)

**特点:** 直观简单,地址固定

### 3. 间接寻址

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>间接寻址</strong>
    <p style="margin: 5px 0;">指令给出的是操作数地址的地址。</p>
</div>

**格式:** OP ((ADDR))

**特点:** 寻址范围大,速度慢

### 4. 寄存器寻址

!!! info "寄存器寻址"
    操作数存放在寄存器中。

**格式:** OP R

**示例:** MOV AX, BX

**特点:** 速度快,无需访问内存

### 5. 寄存器间接寻址

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>寄存器间接寻址</strong>
    <p style="margin: 5px 0;">寄存器中存放操作数的内存地址。</p>
</div>

**格式:** OP (R)

**示例:** MOV AX, (BX)

**特点:** 寻址范围大,比间接寻址快

### 6. 变址寻址

!!! warning "变址寻址"
    有效地址 = 变址寄存器内容 + 形式地址

**格式:** OP (R + D)

**示例:** MOV AX, 100(BX)

**应用:** 数组访问

### 7. 基址寻址

<div style="background-color: #F3E5F5; padding: 15px; margin: 10px 0; border-left: 4px solid #9C27B0; border-radius: 5px;">
    <strong>基址寻址</strong>
    <p style="margin: 5px 0;">有效地址 = 基址寄存器内容 + 形式地址</p>
</div>

**应用:** 多道程序设计

### 8. 相对寻址

!!! success "相对寻址"
    有效地址 = PC内容 + 形式地址

**应用:** 转移指令
