# 指令集架构

## 概述

!!! note "指令集架构(ISA)"
    指令集架构是软硬件之间的抽象层,定义了处理器能执行的所有指令。

## 主要指令集架构

### x86架构

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>x86架构</strong>
    <p style="margin: 5px 0;">Intel开发的CISC架构,主导PC市场。</p>
</div>

**特点:**

- CISC架构
- 向后兼容
- 变长指令
- 代表: Intel、AMD处理器

### ARM架构

!!! tip "ARM架构"
    ARM公司开发的RISC架构,主导移动市场。

**特点:**

- RISC架构
- 低功耗
- 固定指令长度
- 代表: 手机、嵌入式设备

### MIPS架构

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>MIPS架构</strong>
    <p style="margin: 5px 0;">经典的RISC架构,教学常用。</p>
</div>

**特点:**

- RISC架构
- 简洁优雅
- 教学和研究

### RISC-V架构

!!! info "RISC-V架构"
    开源的RISC架构,发展迅速。

**特点:**

- 开源免费
- 模块化设计
- 可扩展
- 未来发展潜力大
