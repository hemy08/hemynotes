# 应用软件分类

## 概述

应用软件是为了解决特定问题而开发的软件,利用系统软件提供的接口访问系统资源。应用软件种类繁多,覆盖了各个领域。

## 办公软件

!!! note "办公软件"
    办公软件是日常工作中最常用的应用软件。

### 1. 文字处理软件

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>文字处理软件</strong>
    <p style="margin: 5px 0;">用于创建、编辑和格式化文本文档。</p>
</div>

**主要功能:**

- 文字输入和编辑
- 格式设置(字体、段落、页面)
- 图文混排
- 表格制作
- 拼写检查
- 文档打印

**代表软件:**

- Microsoft Word
- WPS文字
- LibreOffice Writer
- Google Docs

### 2. 电子表格软件

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>电子表格软件</strong>
    <p style="margin: 5px 0;">用于数据处理、计算和分析。</p>
</div>

**主要功能:**

- 数据输入和编辑
- 公式和函数计算
- 图表制作
- 数据排序和筛选
- 数据透视表
- 数据分析

**常用函数:**

```
SUM(A1:A10)        // 求和
AVERAGE(B1:B10)    // 平均值
MAX(C1:C10)        // 最大值
MIN(D1:D10)        // 最小值
IF(E1>60,"及格","不及格")  // 条件判断
VLOOKUP(F1,A:B,2,FALSE)   // 查找
```

**代表软件:**

- Microsoft Excel
- WPS表格
- LibreOffice Calc
- Google Sheets

### 3. 演示文稿软件

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>演示文稿软件</strong>
    <p style="margin: 5px 0;">用于制作演示幻灯片。</p>
