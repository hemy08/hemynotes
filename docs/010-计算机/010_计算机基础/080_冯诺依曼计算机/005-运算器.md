# 冯诺依曼计算机的运算器

## 概述

运算器是冯诺依曼计算机的执行单元,负责执行各种算术运算和逻辑运算。

## 运算器的功能

!!! note "运算器功能"
    运算器的主要功能包括:

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>运算器功能</strong>
    <ul style="margin: 5px 0;">
        <li>算术运算: 加、减、乘、除</li>
        <li>逻辑运算: 与、或、非、异或</li>
        <li>移位操作: 左移、右移</li>
        <li>比较操作: 等于、大于、小于</li>
    </ul>
</div>

## 运算器的组成

```mermaid
graph TB
    A[运算器] --> B[算术逻辑单元<br/>ALU]
    A --> C[累加器<br/>ACC]
    A --> D[通用寄存器]
    A --> E[状态寄存器<br/>PSW]
```

### 1. 算术逻辑单元(ALU)

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>算术逻辑单元(Arithmetic Logic Unit, ALU)</strong>
    <p style="margin: 5px 0;">执行算术和逻辑运算的核心部件。</p>
</div>

**功能:**

- 算术运算
- 逻辑运算
- 移位操作
- 比较操作

### 2. 累加器(ACC)

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>累加器(Accumulator, ACC)</strong>
    <p style="margin: 5px 0;">存放运算结果和操作数。</p>
</div>

**功能:**

- 存放操作数
- 存放运算结果
- 参与运算

### 3. 通用寄存器

<div style="background-color: #F3E5F5; padding: 15px; margin: 10px 0; border-left: 4px solid #9C27B0; border-radius: 5px;">
    <strong>通用寄存器</strong>
    <p style="margin: 5px 0;">存放操作数和中间结果。</p>
</div>

**常见寄存器:**

- AX, BX, CX, DX (x86)
- R0, R1, R2, ... (RISC)

### 4. 状态寄存器(PSW)

<div style="background-color: #FCE4EC; padding: 15px; margin: 10px 0; border-left: 4px solid #E91E63; border-radius: 5px;">
    <strong>状态寄存器(Program Status Word, PSW)</strong>
    <p style="margin: 5px 0;">存放运算结果的状态标志。</p>
</div>

**常见标志位:**

- ZF: 零标志
- SF: 符号标志
- CF: 进位标志
- OF: 溢出标志

### 2. 逻辑运算

!!! tip "逻辑运算"
    执行逻辑运算。

**运算类型:**

- AND: 逻辑与
- OR: 逻辑或
- NOT: 逻辑非
- XOR: 逻辑异或

### 3. 移位操作

!!! tip "移位操作"
    对数据进行移位操作。

**移位类型:**

- SHL: 左移
- SHR: 逻辑右移
- SAR: 算术右移
- ROL: 循环左移
- ROR: 循环右移
