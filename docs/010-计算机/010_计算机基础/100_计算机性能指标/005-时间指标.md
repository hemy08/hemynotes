# 时间指标

## 概述

时间指标是衡量计算机响应速度的重要参数,包括响应时间、执行时间等。

## 响应时间

!!! note "响应时间"
    响应时间是从用户发送请求到系统做出响应的时间。

### 定义

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>响应时间定义</strong>
    <p style="margin: 5px 0;">从用户向计算机发送一个请求,到系统对该请求做出响应并获得所需结果的等待时间。</p>
</div>

### 响应时间的组成

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>响应时间组成</strong>
    <ul style="margin: 5px 0;">
        <li>CPU时间: 程序执行时间</li>
        <li>等待时间: I/O等待、资源等待</li>
        <li>系统开销: 操作系统开销</li>
    </ul>
</div>

**公式:**

```
响应时间 = CPU时间 + 等待时间 + 系统开销
```

## 执行时间

!!! tip "执行时间"
    执行时间是程序实际占用CPU的时间。

### 定义

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>执行时间定义</strong>
    <p style="margin: 5px 0;">执行时间 = 指令数 × CPI × 时钟周期</p>
</div>

### 执行时间的计算

**示例:**

- 指令数: 100万条
- CPI: 2
- 时钟周期: 1ns
- 执行时间: 100万 × 2 × 1ns = 2ms

## CPU时间

!!! info "CPU时间"
    CPU时间是程序占用CPU的时间。

### 用户CPU时间

<div style="background-color: #F3E5F5; padding: 10px; margin: 10px 0; border-left: 4px solid #9C27B0;">
    <strong>用户CPU时间</strong>
    <p style="margin: 5px 0;">用户程序执行时间。</p>
</div>

### 系统CPU时间

<div style="background-color: #FCE4EC; padding: 10px; margin: 10px 0; border-left: 4px solid #E91E63;">
    <strong>系统CPU时间</strong>
    <p style="margin: 5px 0;">操作系统执行时间。</p>
</div>

**总CPU时间:**

```
总CPU时间 = 用户CPU时间 + 系统CPU时间
```

## 吞吐率

!!! success "吞吐率"
    吞吐率是系统在单位时间内处理的请求数量。

### 定义

<div style="background-color: #E0F2F1; padding: 15px; margin: 10px 0; border-left: 4px solid #00BCD4; border-radius: 5px;">
    <strong>吞吐率定义</strong>
    <p style="margin: 5px 0;">吞吐率 = 完成的请求数 / 总时间</p>
</div>

### 吞吐率与响应时间的关系

<div style="border: 2px solid #4CAF50; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>吞吐率与响应时间</strong>
    <p style="margin: 5px 0;">吞吐率 ≈ 1 / 平均响应时间</p>
</div>

## 周转时间

!!! warning "周转时间"
    周转时间是作业从提交到完成的时间。

### 定义

<div style="background-color: #FFF9C4; padding: 15px; margin: 10px 0; border-left: 4px solid #FFC107; border-radius: 5px;">
    <strong>周转时间定义</strong>
    <p style="margin: 5px 0;">周转时间 = 完成时间 - 提交时间</p>
</div>

### 平均周转时间

<div style="border: 2px solid #2196F3; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>平均周转时间</strong>
    <p style="margin: 5px 0;">平均周转时间 = 总周转时间 / 作业数</p>
</div>
