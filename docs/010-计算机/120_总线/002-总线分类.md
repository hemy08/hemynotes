# 总线分类

## 概述

!!! note "总线分类"
    根据连接部件、功能和位置对总线进行分类。

## 按功能分类

### 数据总线

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>数据总线</strong>
    <p style="margin: 5px 0;">传输数据信息,双向传输。</p>
</div>

**特点:**

- 双向传输
- 宽度决定数据传输能力
- 常见宽度: 8/16/32/64位

### 地址总线

!!! tip "地址总线"
    传输地址信息,单向传输。

**特点:**

- 单向传输(CPU→其他)
- 宽度决定寻址范围
- 32位可寻址4GB空间

### 控制总线

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>控制总线</strong>
    <p style="margin: 5px 0;">传输控制信号和状态信号。</p>
</div>

**信号类型:**

- 时钟信号
- 读写控制
- 中断请求
- 总线请求/响应

## 按位置分类

!!! info "按位置分类"
    - **片内总线**: 芯片内部总线
    - **系统总线**: 连接主要部件
    - **通信总线**: 系统间通信

## 按数据传输方式分类

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>传输方式分类</strong>
</div>

### 串行总线

- 逐位传输数据
- 线路少,成本低
- 适合长距离传输
- 示例: USB、SATA

### 并行总线

!!! warning "并行总线"
    多位同时传输。

- 传输速度快
- 线路多,成本高
- 适合短距离传输
- 示例: PCI、IDE
