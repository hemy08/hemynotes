# 总线标准

## 概述

!!! note "总线标准"
    总线的规范和标准,确保不同设备的兼容性和互操作性。

## 系统总线标准

### ISA总线

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>ISA(工业标准结构)</strong>
    <p style="margin: 5px 0;">早期PC总线标准。</p>
</div>

- 数据宽度: 8/16位
- 工作频率: 8MHz
- 已淘汰

### PCI总线

!!! tip "PCI总线"
    现代PC常用总线标准。

**特点:**

- 数据宽度: 32/64位
- 工作频率: 33/66MHz
- 支持即插即用
- 支持多主设备

### PCI-E总线

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>PCI Express</strong>
    <p style="margin: 5px 0;">高速串行总线标准。</p>
</div>

**特点:**

- 串行传输
- 点对点连接
- 高带宽
- 支持热插拔

## 外部总线标准

### USB

!!! info "USB(通用串行总线)"
    最常用的外部总线。

**版本:**

- USB 1.1: 12Mbps
- USB 2.0: 480Mbps
- USB 3.0: 5Gbps
- USB 3.1: 10Gbps

### SATA

<div style="background-color: #FFF3E0; padding: 15px; margin: 10px 0; border-left: 4px solid #FF9800; border-radius: 5px;">
    <strong>SATA(串行ATA)</strong>
    <p style="margin: 5px 0;">硬盘接口标准。</p>
</div>

**版本:**

- SATA 1.0: 150MB/s
- SATA 2.0: 300MB/s
- SATA 3.0: 600MB/s
