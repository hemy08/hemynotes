# 进程同步

## 概述

!!! note "进程同步"
    协调多个进程的执行顺序,保证数据一致性。

## 同步机制

### 信号量

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>信号量</strong>
    <p style="margin: 5px 0;">整型变量,用于控制进程同步。</p>
</div>

**操作:**

- **P操作**: wait,申请资源
- **V操作**: signal,释放资源

### 互斥锁

!!! tip "互斥锁"
    保证临界区互斥访问。

**操作:**

- lock: 加锁
- unlock: 解锁

### 条件变量

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>条件变量</strong>
    <p style="margin: 5px 0;">等待特定条件成立。</p>
</div>

**操作:**

- wait: 等待条件
- signal: 通知条件成立
