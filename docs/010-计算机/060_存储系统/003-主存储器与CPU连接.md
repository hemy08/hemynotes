# 主存储器与CPU连接

## 概述

!!! note "存储器与CPU连接"
    主存储器通过地址总线、数据总线和控制总线与CPU连接,实现数据交换。

## 连接方式

### 地址总线连接

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>地址总线</strong>
    <p style="margin: 5px 0;">CPU通过地址总线向存储器发送访问地址。</p>
</div>

**地址线数量:** 决定寻址范围

**计算:** 寻址范围 = 2^地址线数量

### 数据总线连接

!!! tip "数据总线"
    CPU与存储器之间传输数据的通道。

**数据线数量:** 决定一次传输的数据量

**常见宽度:** 8位、16位、32位、64位

### 控制总线连接

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>控制总线</strong>
    <p style="margin: 5px 0;">传输控制信号,协调CPU与存储器工作。</p>
</div>

**主要控制信号:**

- **读信号(READ)**: 存储器读操作
- **写信号(WRITE)**: 存储器写操作
- **片选信号(CS)**: 选择存储芯片

## 存储器接口

!!! info "存储器接口"
    存储器与CPU之间的接口电路。

**组成:**

- 地址译码器
- 读写控制逻辑
- 数据缓冲器
