# 存储器扩展

## 概述

!!! note "存储器扩展"
    通过多片存储芯片组合,扩展存储器的容量和字长。

## 位扩展

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>位扩展</strong>
    <p style="margin: 5px 0;">增加存储字长,数据线并联。</p>
</div>

**方法:**

- 地址线并联
- 数据线分别连接
- 片选信号并联
- 读写信号并联

**示例:** 用8片1K×1位组成1K×8位

## 字扩展

!!! tip "字扩展"
    增加存储单元数量,地址线高位用于片选。

**方法:**

- 地址线低位并联
- 地址线高位译码产生片选
- 数据线并联
- 读写信号并联

**示例:** 用8片1K×8位组成8K×8位

## 字位同时扩展

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>字位同时扩展</strong>
    <p style="margin: 5px 0;">既增加字长又增加存储单元数量。</p>
</div>

**方法:** 先位扩展,再字扩展

**示例:** 用8片1K×4位组成4K×8位
