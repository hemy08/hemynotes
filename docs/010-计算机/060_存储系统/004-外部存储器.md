# 外部存储器

## 概述

!!! note "外部存储器"
    外部存储器(辅存)用于永久存储大量数据,容量大但速度慢,CPU不能直接访问。

## 硬盘存储器

<div style="background-color: #E3F2FD; padding: 15px; margin: 10px 0; border-left: 4px solid #2196F3; border-radius: 5px;">
    <strong>硬盘(HDD)</strong>
    <p style="margin: 5px 0;">利用磁记录技术存储数据的机械设备。</p>
</div>

### 硬盘结构

- **磁头**: 读写数据
- **磁道**: 同心圆轨道
- **柱面**: 所有盘面相同磁道
- **扇区**: 磁道的分段,基本存储单位

### 性能指标

!!! tip "硬盘性能"
    - **容量**: 存储容量
    - **转速**: 5400/7200/10000/15000 RPM
    - **平均寻道时间**: 5-10ms
    - **数据传输率**: 100-300 MB/s

## 固态硬盘

<div style="background-color: #E8F5E9; padding: 15px; margin: 10px 0; border-left: 4px solid #4CAF50; border-radius: 5px;">
    <strong>固态硬盘(SSD)</strong>
    <p style="margin: 5px 0;">基于闪存技术的存储设备,无机械结构。</p>
</div>

### SSD特点

- **优点**: 速度快、无噪音、抗震
- **缺点**: 成本高、寿命有限
- **接口**: SATA、PCIe、NVMe

## 光盘存储器

!!! info "光盘"
    利用激光技术存储数据。

**类型:**

- **CD**: 700MB
- **DVD**: 4.7GB(单面单层)
- **蓝光光盘**: 25GB(单面单层)
