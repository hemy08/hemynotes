# 计算机系统

## 概述

计算机系统是由硬件子系统和软件子系统组成的复杂系统,是能够自行按照已设定的程序进行数据处理的电子设备。

!!! note "计算机系统定义"
    计算机系统包括硬件子系统和软件子系统,是软件与硬件相结合、面向系统、侧重应用的自动化求解工具。

## 计算机发展历程

!!! tip "计算机发展历程"
    计算机经历了四代发展:

### 第一代(1946-1957): 电子管

<div style="border: 2px solid #4CAF50; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>第一代计算机</strong>
    <ul style="margin: 5px 0;">
        <li>使用电子管作为逻辑元件</li>
        <li>面向机器的语言</li>
        <li>体积大、功耗高、可靠性差</li>
    </ul>
</div>

### 第二代(1958-1964): 晶体管

<div style="border: 2px solid #2196F3; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>第二代计算机</strong>
    <ul style="margin: 5px 0;">
        <li>使用晶体管作为逻辑元件</li>
        <li>面向过程的语言</li>
        <li>有了操作系统的雏形</li>
    </ul>
</div>

### 第三代(1964-1971): 中小规模集成电路

<div style="border: 2px solid #FF9800; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>第三代计算机</strong>
    <ul style="margin: 5px 0;">
        <li>使用中小规模集成电路</li>
        <li>高级语言迅速发展</li>
        <li>有了分时操作系统</li>
    </ul>
</div>

### 第四代(1972-现在): 大规模、超大规模集成电路

<div style="border: 2px solid #9C27B0; padding: 10px; margin: 10px 0; border-radius: 5px;">
    <strong>第四代计算机</strong>
    <ul style="margin: 5px 0;">
        <li>使用大规模、超大规模集成电路</li>
        <li>微处理器出现</li>
        <li>并行、流水线、高速缓存、虚拟存储器</li>
    </ul>
</div>

## 重要概念

### 机器字长

!!! info "机器字长"
    计算机一次整数运算所能处理的二进制位数。

**意义:**

- 决定计算精度
- 影响运算速度
- 决定内存容量上限

### 摩尔定律

!!! warning "摩尔定律"
    集成电路上可容纳的晶体管数目约每隔18个月会增长一倍,整体性能也将提升一倍。

**意义:**

- 揭示了信息技术进步的速度
- 指导计算机产业发展
- 预测技术发展趋势

## 学习建议

!!! success "学习建议"
    建议按照以下顺序学习:
    1. **理解系统概念**: 了解计算机系统的基本组成  
    2. **掌握硬件知识**: 学习硬件子系统的工作原理  
    3. **了解软件系统**: 理解软件子系统的分类和功能  
    4. **深入操作系统**: 学习操作系统的核心功能
