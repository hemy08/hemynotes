### 🔧 技术改进

#### 标签页格式标准化

**缩进规则**:
```
=== "语言名称"        ← 标签页标题
                      ← 空行
    ```语言标识       ← 4空格 + 代码块开始
    代码内容           ← 4空格缩进
    ```               ← 4空格 + 代码块结束
```

**语言标识统一**:
- C → `c`
- C++ → `cpp`
- Python → `python`
- Java → `java`
- Go → `go`
- Rust → `rust`

#### 代码风格统一

**命名规范**:
- C: snake_case (如 `next_greater_element`)
- C++: camelCase (如 `nextGreaterElement`)
- Python: snake_case (如 `next_greater_element`)
- Java: camelCase (如 `nextGreaterElement`)
- Go: PascalCase (如 `NextGreaterElement`)
- Rust: snake_case (如 `next_greater_element`)

**注释风格**:
```c
// C风格: 单行注释
/* 多行注释 */
```

```cpp
// C++风格: 单行注释
/** 文档注释 */
```

```python
# Python风格
"""
文档字符串
"""
```

```java
// Java风格
/**
 * Javadoc注释
 */
```

```go
// Go风格
// 文档注释
```

```rust
// Rust风格
/// 文档注释
```
